## The print() Function

The `print()` function is one of the most useful tools in Python.
It tells the computer: **"Show this text on the screen."**

### How It Works:

```python
print("Hello, World!")
```

Let's break it down:
- `print` -- This is the name of the **function** (a command that does something)
- `(` and `)` -- These **parentheses** wrap around what you want to print
- `"Hello, World!"` -- This is a **string** (text inside quotation marks)

When you run this, the computer displays:
```
Hello, World!
```

### Important Rules:
1. You need the parentheses `()` -- they are required!
2. Text must be inside quotes: `"like this"` or `'like this'`
3. Python is case-sensitive: `print` works, but `Print` does NOT

---

## Key Vocabulary

- **print()**: A function that displays text on the screen
- **String**: Text inside quotation marks, like `"hello"`
- **Function**: A command that tells the computer to do something
- **Syntax**: The rules for how code must be written
