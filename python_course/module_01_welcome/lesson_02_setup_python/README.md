# Module 1, Lesson 2: Setting Up Python

## Your Mission Today
Get Python installed on your computer so you can start writing code!

## Learning Objectives
By the end of this lesson, you will be able to:
- Install Python on your computer
- Open Python's built-in editor (IDLE)
- Know the difference between a code file and the interactive shell

---

## Step 1: Download Python

1. Go to [python.org](https://www.python.org/downloads/)
2. Click the big yellow button that says **"Download Python 3.x.x"**
3. Run the installer

### IMPORTANT for Windows Users:
When the installer opens, CHECK the box that says:
**"Add Python to PATH"**
This is very important! Do not skip it!

Then click "Install Now."

### For Mac Users:
Python might already be on your computer! But we want the newest version.
Download it from the link above and run the installer.

---

## Step 2: Check That Python Works

After installing, let's make sure it works!

### On Windows:
1. Press the Windows key
2. Type `cmd` and press Enter
3. In the black window that appears, type: `python --version`
4. You should see something like: `Python 3.12.1`

### On Mac:
1. Open the "Terminal" app (search for it in Spotlight)
2. Type: `python3 --version`
3. You should see something like: `Python 3.12.1`

If you see a version number, Python is installed! Hooray!

---

## Step 3: Meet IDLE

**IDLE** is a simple code editor that comes with Python.
It is where we will write and run our programs.

### To open IDLE:
- **Windows**: Search for "IDLE" in the Start menu
- **Mac**: Search for "IDLE" in Spotlight, or find it in Applications > Python

When IDLE opens, you will see the **Python Shell**. It looks like this:

```
Python 3.12.1 (...)
Type "help" for more information.
>>>
```

The `>>>` is called the **prompt**. It means Python is waiting for you
to type something!

---

## Step 4: Try It Out!

In the IDLE shell, type this and press Enter:

```python
print("Python is working!")
```

If you see `Python is working!` appear, everything is set up correctly!

---

## Key Vocabulary

- **Install**: Putting a program on your computer so you can use it
- **IDLE**: Python's built-in code editor (it stands for Integrated Development and Learning Environment)
- **Shell**: A window where you can type Python code and see results immediately
- **Prompt (>>>)**: The symbol that means Python is ready for your next instruction

---

## Recommended Editors (Optional)

IDLE is great for starting, but as you grow, you might enjoy:
- **Thonny** (thonny.org) -- Designed for beginners, very friendly
- **VS Code** (code.visualstudio.com) -- Used by professionals, lots of features

For this course, IDLE or Thonny will work perfectly!
