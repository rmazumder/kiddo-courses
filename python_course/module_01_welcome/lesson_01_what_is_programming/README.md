## What Is a Program?

A **program** is a set of instructions that tells a computer what to do.

Think of it like a recipe:
- A recipe tells a cook what steps to follow to make a cake.
- A program tells a computer what steps to follow to do a task.

**Example -- A "recipe" for making a peanut butter sandwich:**
1. Get two slices of bread.
2. Open the peanut butter jar.
3. Use a knife to spread peanut butter on one slice.
4. Put the other slice on top.
5. Done! Enjoy your sandwich.

A computer program works the same way -- it is a list of clear, step-by-step
instructions. The computer reads them from top to bottom and does exactly
what you say.

## Why Python?

There are many programming languages (like English, Spanish, and French
are different human languages). Some popular ones are:
- **Python** -- Easy to read, great for beginners
- **JavaScript** -- Used to make websites interactive
- **Scratch** -- A visual block-based language for kids

We are learning **Python** because:
1. It reads almost like English
2. It is used by real professionals (at Google, NASA, Netflix!)
3. It is great for games, science, art, and more
4. It is FREE!

## Key Vocabulary

- **Program**: A set of instructions for a computer
- **Programming**: The act of writing programs
- **Code**: The text that makes up a program
- **Python**: The programming language we are learning
- **Bug**: A mistake in your code (we all make them -- it is okay!)
