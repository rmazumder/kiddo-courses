---

## Common Loop Patterns

These patterns appear in almost every program. Once you learn them,
you will recognize them everywhere!

1. **Counter**: Count how many times something happens
2. **Accumulator**: Add up a running total
3. **Search**: Look for something specific
4. **Nested Loop**: A loop inside another loop
