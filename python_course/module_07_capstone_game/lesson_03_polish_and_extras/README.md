# Module 7, Lesson 3: Polish and Extras

## Your Mission Today
Learn techniques to make your game even better!

## Learning Objectives
By the end of this lesson, you will be able to:
- Add a save/load feature concept
- Add color and formatting to make the game look good
- Add more game features like a scoring system, timer, or random events

---

## Ways to Improve Your Game

### 1. Better Text Formatting
- Use divider lines to separate sections
- Add ASCII art for rooms or monsters
- Use consistent formatting for game messages

### 2. More Game Features
- **Score system**: Points for finding items, solving puzzles
- **Time limit**: Track how many turns the player uses
- **Random events**: Random encounters or surprises
- **Multiple endings**: Different endings based on player choices
- **Difficulty levels**: Easy, Medium, Hard

### 3. Better Error Handling
- Handle unexpected input gracefully
- Give helpful suggestions when the player is stuck

### 4. Code Organization
- Keep your code organized with functions
- Add comments so others can understand your code
- Test every path through the game
