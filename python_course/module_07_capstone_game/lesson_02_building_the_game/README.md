# Module 7, Lesson 2: Building the Game

## Your Mission Today
Start building your text-based adventure game, piece by piece!

## Learning Objectives
By the end of this lesson, you will be able to:
- Set up the game's data structures (rooms, player, items)
- Create the main game loop
- Handle player commands (move, look, take, inventory, help)

---

## Building Block by Block

We will build the game in steps:

1. **Step 1**: Set up the rooms and player data
2. **Step 2**: Create a function to display the current room
3. **Step 3**: Create a function to handle movement
4. **Step 4**: Create a function to pick up items
5. **Step 5**: Build the main game loop
6. **Step 6**: Add win/lose conditions

---

## Activity

Open `game_starter.py` to see a WORKING example game.
Study it, run it, and understand how each part works.

Then start building your OWN game using the template
or modifying the starter!
