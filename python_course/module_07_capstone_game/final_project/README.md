# Module 7: FINAL PROJECT

## Your Capstone Project

This is it -- the moment everything has been building toward!

**Your mission**: Build your own text-based adventure game!

## Requirements for Your Game

Your game MUST include:

1. **At least 5 rooms/locations** with descriptions
2. **At least 3 items** the player can pick up
3. **Player movement** between rooms (north, south, east, west, etc.)
4. **An inventory system** (take items, view inventory)
5. **A win condition** (how does the player beat the game?)
6. **A lose condition** (how can the player fail?)
7. **At least 3 functions** you wrote yourself
8. **A main game loop** using while

## Bonus Features (for extra XP!)

- ASCII art for rooms or characters
- Random events
- A scoring system
- A health system
- Puzzles or riddles
- Multiple endings
- A help command

## How to Submit

1. Build your game in `my_game.py`
2. Test it thoroughly -- play through every path!
3. Make sure it runs without errors
4. Show it to a friend or family member!

## Earning Your Badge

When your game is complete and working, you earn:

- **+100 XP** for completing the capstone project
- **[G] Game Creator badge**
- **PYTHON QUEST CHAMPION** title (if you earned all 7 badges!)
