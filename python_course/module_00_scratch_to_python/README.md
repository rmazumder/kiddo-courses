# Module 0: From Scratch to Python!

## Welcome, Visual Coder!

Have you used **Scratch** before? Those colorful blocks you snap together to make
sprites move, talk, and play sounds? That is real programming!

**Python does the exact same things** - but instead of clicking and snapping blocks,
you *type* the instructions. That is the only difference. Everything you learned in
Scratch carries right over to Python.

---

## Scratch vs Python: Side-by-Side

| Scratch Concept | Scratch Block | Python Equivalent |
|---|---|---|
| Start the program | "when green flag clicked" | `if __name__ == "__main__":` |
| Make something say text | "say Hello for 2 secs" | `print("Hello")` |
| Move a character | "move 10 steps" | `x += 10` |
| Repeat something | "repeat 10" | `for i in range(10):` |
| Check a condition | "if touching color" | `if condition:` |
| Store a value | "set x to 0" | `x = 0` |
| Pause / wait | "wait 1 secs" | `time.sleep(1)` |
| Loop forever | "forever" | `while True:` |
| Send a signal | "broadcast message" | calling a function |
| Your own variable | "my variable" | `my_variable = value` |
