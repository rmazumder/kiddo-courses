## Hook: Building with LEGO

build a cool spaceship, you can write down the steps and build it
again and again without figuring it out from scratch.

A function is a set of instructions that you give a name. Once
defined, you can use it as many times as you want!

---

## What Is a Function?

A **function** is a reusable block of code that does a specific job.

You have already used functions:
- `print()` -- displays text
- `len()` -- counts characters
- `input()` -- gets user input
- `range()` -- creates a sequence of numbers

Those are **built-in functions** (Python made them for you).
Now you will learn to make your **own** functions!

---

## Creating a Function

```python
def greet():
    print("Hello there!")
    print("Welcome to Python Quest!")
```

- `def` -- this keyword means "I am DEFining a new function"
- `greet` -- this is the NAME of your function
- `()` -- parentheses (we will put parameters here later)
- `:` -- the colon starts the function body
- The indented lines are the BODY (what the function does)

IMPORTANT: Defining a function does NOT run it! You must CALL it:

```python
greet()   # This CALLS the function -- now it runs!
```
