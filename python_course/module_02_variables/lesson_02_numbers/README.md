## Number Types

Python has two types of numbers:

1. **Integer (int)**: A whole number with no decimal point
   - Examples: `5`, `100`, `-3`, `0`

2. **Float**: A number WITH a decimal point
   - Examples: `3.14`, `0.5`, `-2.7`, `100.0`

---

## Math Operators

| Symbol | What It Does | Example | Result |
|--------|-------------|---------|--------|
| `+` | Addition | `5 + 3` | `8` |
| `-` | Subtraction | `10 - 4` | `6` |
| `*` | Multiplication | `3 * 7` | `21` |
| `/` | Division | `15 / 3` | `5.0` |
| `**` | Power (exponent) | `2 ** 3` | `8` |
| `//` | Floor division (no remainder) | `7 // 2` | `3` |
| `%` | Modulo (remainder only) | `7 % 2` | `1` |
