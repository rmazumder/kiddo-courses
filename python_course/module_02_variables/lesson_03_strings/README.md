## What Is a String?

A **string** is any text inside quotation marks.

```python
greeting = "Hello!"         # Double quotes
name = 'Python'             # Single quotes -- both work!
sentence = "I love coding"  # Spaces are fine inside strings
```

Strings can hold letters, numbers, spaces, and symbols -- anything
you can type on a keyboard!
