## What Is a Boolean?

A **boolean** (say "BOO-lee-un") is a value that is either `True` or `False`.
That is it -- just two possible values!

Think of it like a light switch: it is either ON (True) or OFF (False).

```python
is_raining = True
has_homework = False
```

Booleans are named after George Boole, a mathematician who studied
logic (true/false thinking).

---

## Why Do Booleans Matter?

Booleans let your program make DECISIONS. Coming up in Module 3, you will
learn to write code like:

```python
if is_raining:
    print("Bring an umbrella!")
```
