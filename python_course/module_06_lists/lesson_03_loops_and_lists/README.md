## Looping Through Lists

This is one of the most common patterns in all of programming:

```python
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}!")
```

The variable `fruit` takes on each value in the list, one at a time.
