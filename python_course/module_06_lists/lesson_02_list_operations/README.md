# Module 6, Lesson 2: List Operations

## Your Mission Today
Learn how to add, remove, and sort items in lists!

## Learning Objectives
By the end of this lesson, you will be able to:
- Add items with `.append()` and `.insert()`
- Remove items with `.remove()` and `.pop()`
- Sort lists with `.sort()`
- Combine lists

---

## Key List Methods

| Method | What It Does | Example |
|--------|-------------|---------|
| `.append(item)` | Add item to the end | `fruits.append("grape")` |
| `.insert(pos, item)` | Insert at a specific position | `fruits.insert(0, "mango")` |
| `.remove(item)` | Remove the first occurrence | `fruits.remove("banana")` |
| `.pop()` | Remove and return the last item | `last = fruits.pop()` |
| `.pop(pos)` | Remove and return item at position | `first = fruits.pop(0)` |
| `.sort()` | Sort in order (A-Z or small to big) | `fruits.sort()` |
| `.reverse()` | Reverse the list | `fruits.reverse()` |
| `.count(item)` | Count occurrences | `fruits.count("apple")` |
| `.index(item)` | Find position of an item | `fruits.index("banana")` |
