📚 Key Words — Learn These!
🔌 BT
Bluetooth
A short-range wireless radio link. Sends data between your phone and the HC-05 module — like texting but for robots!
🖥 SS
SoftwareSerial
An Arduino library that turns any two pins into an extra serial port. Lets us talk to HC-05 while keeping USB free for uploading!
⚡ VD
Voltage Divider
Two resistors that split voltage. Converts Arduino's 5V signal down to ~3.3V to safely protect the HC-05 RX pin.
🔄 TX/RX
Serial Data
Information sent one bit at a time in a line. TX = transmit, RX = receive. HC-05 uses serial to talk to Arduino wirelessly!
📶 How Bluetooth Control Works
[Your Phone] [HC-05 Module] [Arduino] | | | |--- press F (Forward) ----> |--- serial data ---> |--- forward(200) | | | |--- press B (Backward) ---> |--- serial data ---> |--- backward(200) | | | |--- press S (Stop) -------> |--- serial data ---> |--- stopRobot() | | | |--- press H (Horn) -------> |--- serial data ---> |--- tone(800Hz) | | | Fast blink = searching | Slow blink = connected!
🔋 HC-05 LED Signals
Fast blink — searching for phone
Slow blink — connected & ready!
📱 Pairing Steps
Power on robot → Bluetooth settings
Find "HC-05" → Password: 1234
Your phone sends a single letter — the Arduino reads it and moves the robot. It's like texting commands to your robot!
⚡ Wiring the HC-05 & Command Reference
🔌 HC-05 Wiring Diagram
HC-05 VCC ---> Arduino 5V HC-05 GND ---> Arduino GND HC-05 TXD ---> Arduino Pin 7 HC-05 RXD <--- Voltage Divider Divider (5V -> 3.3V): Pin 8 --[1k]--+--[2k]-- GND | HC-05 RXD
Why a divider? HC-05 RX needs 3.3V. Arduino sends 5V. The 1k + 2k resistors split it safely!
Note: Disconnect HC-05 TX/RX wires before uploading code!

🔌 My Activity Today — Build Your Bluetooth Robot!
SoftwareSerial bluetooth(7, 8); // RX=pin7, TX=pin8 void loop() { if (bluetooth.available()) { char command = bluetooth.read(); // read one letter switch (command) { case 'F': forward(200); break; // phone sent "F" case 'B': backward(200); break; // phone sent "B" case 'L': turnLeft(200); break; // phone sent "L" case 'R': turnRight(200); break; // phone sent "R" case 'S': stopRobot(); break; // phone sent "S" case 'H': tone(A5,800,300); break; // HONK! } } }
The switch statement is like a traffic cop — each letter gets routed to the right movement function!
✅ Build Steps:
  1. Wire HC-05 VCC → 5V and GND → GND
  2. Wire HC-05 TXD → Arduino Pin 7
  3. Build voltage divider (1k + 2k) from Pin 8 → HC-05 RXD
  4. Upload code (HC-05 TX/RX disconnected!)
  5. Reconnect HC-05 wires, power on robot
  6. Pair phone to HC-05 (password: 1234)
  7. Open Bluetooth app, connect, and drive!
