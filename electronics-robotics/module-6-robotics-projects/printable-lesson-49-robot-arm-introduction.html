📚 Key Words — Learn These!
🤖 DOF
Degree of Freedom (DOF)
Each independent way a robot arm can move. Your shoulder, elbow, and wrist each add one DOF. More DOF = more flexible arm!
⚙ SG90
Servo Motor
A motor that rotates to a precise angle (0–180°) when told to. Like a robot joint that goes exactly where you command it!
🆕 ~
Interpolation
Moving from one position to another in many tiny steps so the motion is smooth and graceful — not jerky.
📈 PWM
PWM Signal
The pulse-width modulation signal Arduino sends to a servo. The width of each pulse tells the servo which angle to go to.
🤖 How a Robot Arm Works — Joints & DOF
Human Arm vs Robot Arm
Human Arm: Robot Arm: [Shoulder] = rotate [Base Servo] = rotate L/R [Elbow] = bend [Shoulder Servo] = reach up/down [Wrist] = tilt [Elbow Servo] = extend/retract [Hand] = grab [Gripper] = open/close
Each joint = 1 Degree of Freedom. Our arm has 3 DOF!
Complete Arm (side view)
[Gripper]--[Arm 2]--[Servo 3]--[Arm 1]--[Servo 2] | [Servo 1] | [BASE PLATE] Servo 1 = Base (rotates left/right) Servo 2 = Shoulder (raises/lowers arm) Servo 3 = Elbow (extends/retracts)
Three servos = three joints = a real working robot arm that can reach, grab, and move objects!
💻 Wiring + Manual Control Code
⚡ Servo Wiring (all 3)
Servo 1 (Base) → Pin 3 Servo 2 (Shoulder) → Pin 5 Servo 3 (Elbow) → Pin 6 Each servo: RED → 5V (external supply!) BROWN → GND SIGNAL → Arduino pin
Power tip: Three servos can draw 300–800 mA when moving. If servos jitter or Arduino resets, use an external 5V supply instead of the USB port!
🎮 Pot Control Code (key lines)
// Map pot (0-1023) → angle (0-180) int baseAngle = map(analogRead(A0), 0,1023, 0,180); baseServo.write(baseAngle); shoulderServo.write(shoulderAngle); elbowServo.write(elbowAngle); delay(20); // smooth movement
Pot wiring: Left pin → GND  |  Right pin → 5V  |  Middle pin → A0, A1, A2. Turn the knob to control each joint!

💡 My Activity Today — Build the Robot Arm!
Build Steps
  1. 1Glue Servo 1 to cardboard base — this rotates the whole arm left/right.
  2. 2Mount Servo 2 on base servo's horn — this raises/lowers the shoulder.
  3. 3Attach two popsicle sticks to Servo 2's horn — this is your upper arm.
  4. 4Mount Servo 3 at the end of the upper arm — this is the elbow.
  5. 5Attach a second arm segment to Servo 3 and add gripper at the tip.
  6. 6Wire all three servos and potentiometers to Arduino per the wiring diagram.
  7. 7Upload potentiometer control code and test each joint separately.
  8. 8Upload pick-and-place code and adjust angles until it grabs a target!
Pick-and-Place Angles
// moveArm(base, shoulder, elbow) moveArm( 45, 60, 120); // hover moveArm( 45, 30, 150); // lower grab delay(500); // GRAB! moveArm( 45, 80, 90); // lift up moveArm(135, 80, 90); // rotate moveArm(135, 40, 130); // lower drop delay(500); // RELEASE!

🔧 Troubleshooting Quick Guide
ProblemQuick Fix
Servos jitterUse external 5V supply for servos
Arduino resetsToo much current — external power needed
Arm is wobblyAdd more hot glue to joints
Pot dead zonesCheck wiring: L=GND, R=5V, Mid=A0/1/2
Wrong servo angleservo.write() takes 0–180 only
Glue joints breakRoughen surfaces before re-gluing
