💡 My Build Today — The 6 Steps of Every Engineer!
  • 1 Plan It (15 min): Choose your project. List your sensors, actuators, and materials. Write down what your robot needs to do. +20 XP
  • 2 Build It (30–45 min): Assemble hardware, wire all connections. Double-check EVERYTHING before powering on. +40 XP
  • 3 Code It (20–30 min): Use the starter code for your project. Customize speed, timing, and threshold settings. Upload and test. +40 XP
  • 4 Test & Tune (15–20 min): Run it multiple times. Adjust values. Fix any bugs. +30 XP
  • 5 Wand Check (10 min): Full robot health check with your Magic Measurement Wand — power, motors, sensors, continuity. +50 XP
  • 6 Demo Day! (10–15 min): Present to family/friends — introduce the robot, name 5 components, run a live demo, explain the code, show a wand measurement! +50 XP
