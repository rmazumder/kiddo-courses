📚 Key Words — Learn These!
🔨 f()
Function
A reusable block of code you give a name to. Write it once, call it anywhere! Like teaching the robot a new word.
🏋 PWM
PWM Speed (0–255)
A number that controls how fast a motor spins. 0 = stopped, 255 = full speed. Higher number = more average voltage.
🔌 L298N
Motor Driver
A chip that lets the Arduino control motor power. It switches voltage direction so motors can go forward or backward.
🔈 ms
Calibration
Adjusting motor speeds to fix drift. If the robot curves left, slow the right motor slightly to drive straight!

💻 Writing a Movement Function — The Pattern
🥊 The Function Recipe — forward() step by step
// Before functions (messy!): digitalWrite(leftIN1, HIGH); digitalWrite(leftIN2, LOW); digitalWrite(rightIN3, HIGH); digitalWrite(rightIN4, LOW); analogWrite(leftENA, 180); analogWrite(rightENB, 180); // 6 lines every time! // After functions (clean!): forward(180); // just 1 line!
Every movement function:
1. Sets IN1/IN2 pins HIGH/LOW for direction
2. Sets IN3/IN4 pins HIGH/LOW for direction
3. Writes PWM speed to ENA and ENB

The speed parameter (0–255):
• 0 = stopped
• 120 = slow
• 180 = medium
• 255 = full power

Tip: Start at 180 — never go below 120 or motors stall!
■ Drive in a Square — Loop Challenge
for (int side = 0; side < 4; side++) { forward(180); // drive one side delay(1500); // <-- tune for length stopRobot(); delay(500); turnRight(180); // 90-degree turn delay(450); // <-- tune for angle stopRobot(); delay(500); } stopRobot();
Tuning Tips:
• forward delay = side length
• turn delay = turn angle
• Too much turn? Decrease delay
• Not enough? Increase delay

// === MOTOR PIN SETUP === int leftENA=9, leftIN1=2, leftIN2=3; int rightENB=10, rightIN3=4, rightIN4=5; void forward(int speed) { digitalWrite(leftIN1, HIGH); digitalWrite(leftIN2, LOW); digitalWrite(rightIN3, HIGH); digitalWrite(rightIN4, LOW); analogWrite(leftENA, speed); analogWrite(rightENB, speed); } void stopRobot() { digitalWrite(leftIN1, LOW); digitalWrite(leftIN2, LOW); digitalWrite(rightIN3, LOW); digitalWrite(rightIN4, LOW); analogWrite(leftENA, 0); analogWrite(rightENB, 0); }

🔬 4 Experiments You Did Today!
A
Drive in a Square
Run the square for-loop and tune the delays.
Small time changes make BIG path changes — that is why robotics engineers tune carefully!
B
Figure-8 Circles
Set one wheel faster than the other for gentle curves.
Differential speed is how real robots (and boats!) steer — no steering wheel needed!
C
Motor Calibration
Add a leftOffset or rightOffset to the speed value.
No two motors are identical — a tiny offset number fixes drift and makes the robot go perfectly straight!
D
Wand Voltage Check
Measure OUT1–OUT2 voltage at PWM 100, 150, 200, 255.
Higher PWM = higher average voltage = faster spin. The Wand proved it with numbers!
