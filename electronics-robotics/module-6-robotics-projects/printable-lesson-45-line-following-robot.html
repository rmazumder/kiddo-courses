📚 Key Words — Learn These!
📷 IR
IR Sensor
A tiny sensor that shines infrared light at the ground and watches how much bounces back. White reflects a lot; black absorbs it all.
🤖 ≡
Autonomous
When a robot makes its own decisions without a remote control. The robot reads sensors and chooses what to do all by itself!
🔄 if
Algorithm
A set of step-by-step rules the robot follows. "If left sensor sees black AND right is white, then turn left."
🎲 ~
PWM / Speed
Pulse Width Modulation — how Arduino controls motor speed using analogWrite(). Higher value (0–255) = faster spin.
💡 How IR Line Sensors See the Track
The Flashlight-and-Mirror Trick
How IR Line Sensors Work: White Surface (reflects): Black Surface (absorbs): SENSOR SENSOR | \ / | | \ | | \ / | | \ | | \/ | | \ | ================ (white) |||||||||| (black tape) IR bounces back! IR gets absorbed! Output: HIGH (1) Output: LOW (0) Voltage: ~4.8 V Voltage: ~0.2 V
The sensor is like a flashlight AND a camera in one! White = mirror (reflects back). Black = black hole (absorbs everything). Your Magic Measurement Wand will show you the exact voltages!
🧠 The Robot's Decision Table
The robot reads both sensors 100 times per second and decides what to do:
Left Sensor Right Sensor What It Means Robot Action
HIGH (white) HIGH (white) Both on the line Go FORWARD
LOW (black) HIGH (white) Drifted RIGHT off line Turn LEFT to correct
HIGH (white) LOW (black) Drifted LEFT off line Turn RIGHT to correct
LOW (black) LOW (black) LOST the line! STOP and search
Remember: if the LEFT sensor sees black it means the robot drifted RIGHT, so steer LEFT to get back. Think of it as the sensor calling "HELP! I fell off the road!"
💻 The Core Line-Following Logic
The loop() function — runs ~100 times per second:
// Read both IR sensors (HIGH = white, LOW = black) int L = digitalRead(leftSensor); // Pin 7 int R = digitalRead(rightSensor); // Pin 8 if (L == HIGH && R == HIGH) { forward(160); // Both on line -> Go straight! } else if (L == LOW && R == HIGH) { turnLeft(140); // Left off line -> Correct LEFT } else if (L == HIGH && R == LOW) { turnRight(140); // Right off line -> Correct RIGHT } else { stopRobot(); // Both off line -> STOP + search }
turnLeft() stops the LEFT wheel and keeps the RIGHT wheel going — a gentle correction. This is different from Lesson 44 spinning in place!

🔌 Wiring the IR Sensors (Step 2)
Left IR Sensor: Right IR Sensor: VCC ---> Arduino 5V VCC ---> Arduino 5V GND ---> Arduino GND GND ---> Arduino GND OUT ---> Arduino Pin 7 OUT ---> Arduino Pin 8 Mounting (bottom-front of robot, ~1.5-2 cm above ground): +-----------------------------+ | | | [Left [Right | | Sensor] Sensor] | | ^ ^ | | +--- gap ---+ | | (tape width apart) | | Direction of travel --> | +-----------------------------+
  1. 1Mount both IR sensors on the FRONT-BOTTOM of the robot, spaced just wider than the black tape.
  2. 2Wire VCC → 5V, GND → GND, and OUT pins to Arduino pins 7 and 8.
  3. 3Upload the sensor TEST sketch and hold the robot over white paper — both sensors should read HIGH.
  4. 4Hold over black tape — both sensors should read LOW. If not, adjust the tiny potentiometer on each sensor.
  5. 5Upload the full line-following code. Place robot on track and press go — enjoy your autonomous robot!
  6. 6Tune: if the robot wobbles, reduce driveSpeed. If it overshoots curves, reduce turnSpeed.
Both sensors should be 1.5–2 cm from the ground. Too high = misses the line. Too low = drags on the floor.

🔬 4 Experiments You Did Today!

Sensor on Different Surfaces
Hold the robot over white, black, and gray surfaces and read the Serial Monitor.
Gray gives an in-between reading — that is why gray floors confuse line-following robots!
B
Wand Voltage Check
Measure the OUT pin voltage on white paper vs. black tape using your Multimeter.
White gives ~4.8 V (HIGH), black gives ~0.2 V (LOW) — the invisible conversation made visible!
C
Oval Track Run
Build a simple oval track with black tape. Place robot on line and let it run!
The robot makes hundreds of micro-corrections per second — that is autonomous navigation!
D
Speed Tuning Challenge
Change driveSpeed from 200 down to 120 and watch what happens at curves.
