📚 Key Words — Learn These!
📝 {}
Sketch
The name for an Arduino program. Every sketch has two required parts: setup() and loop().
🔌 7
Digital Pin
A numbered connection on the Arduino that can be set HIGH (5V) or LOW (0V) — like a tiny on/off switch you control with code.
▶ ms
delay()
A command that pauses the sketch for a set number of milliseconds. 1000 ms = 1 second. Makes the LED wait before changing state.
💡 ⇅
HIGH / LOW
HIGH = 5 volts ON the pin (LED lights up). LOW = 0 volts on the pin (LED off). The Arduino flips between them to blink!
✎ Anatomy of Every Arduino Sketch
☀ setup() — Runs ONCE
void setup() { pinMode(7, OUTPUT); // Runs once at start }
Like getting dressed in the morning. You only do it once to get ready for the day!
☀ Runs at startup only
↻ loop() — Runs FOREVER
void loop() { digitalWrite(7, HIGH); delay(1000); // Repeats forever! }
Like breathing — you do it over and over without stopping, forever!
↻ Loops until power off
💡 The Blink Sketch — Full Code & Circuit
Wiring Diagram:
Arduino Uno Breadboard +---------+ | Pin 7 |--[wire]--+ | | [330Ω resistor] | | LED+ (long leg) | | LED- (short leg) | GND |--[wire]--+ +---------+ Row 10: Pin7 wire → resistor leg1 Row 15: resistor leg2 → LED + leg Row 16: LED - leg → wire → GND
The resistor protects the LED — just like in Module 1!
The Blink Sketch:
// Lesson 26: Blink LED on Pin 7 int ledPin = 7; void setup() { pinMode(ledPin, OUTPUT); } void loop() { digitalWrite(ledPin, HIGH); delay(1000); // wait 1s digitalWrite(ledPin, LOW); delay(1000); // wait 1s }
HIGH = 5V (LED ON)  ↔  LOW = 0V (LED OFF) — Arduino flips pin 7 like a tiny light switch!
🔬 Three Commands You Must Know
Command What It Does Real-World Analogy
pinMode(pin, OUTPUT) Tells Arduino: "this pin will send signals" Labeling a door as EXIT
digitalWrite(pin, HIGH) Sets pin to 5V — turns LED ON Flipping a light switch ON
digitalWrite(pin, LOW) Sets pin to 0V — turns LED OFF Flipping a light switch OFF
delay(1000) Waits 1 second (1000 ms) before next line Saying "pause for a moment"

⚡ My Activity Today!
💡 Wiring Steps:
  1. 1Place LED on breadboard (long leg = +, short leg = −)
  2. 2Connect 330Ω resistor from Arduino Pin 7 to LED long leg (+)
  3. 3Connect wire from LED short leg (−) to Arduino GND
💻 Upload Steps:
  1. 1Open Arduino IDE, type the Blink sketch
  2. 2Click the Upload button (right arrow ►)
  3. 3Wait for "Done uploading" — watch it blink!
✅ Components I Used:
  • Arduino Uno
  • USB cable
  • Breadboard
  • LED (any color)
  • 330Ω resistor
  • 3 jumper wires
  • Multimeter
  • Arduino IDE
Multimeter tip: set to DC Voltage, black probe to GND, red probe to Pin 7. Watch it jump between 0V and 5V as the LED blinks!

🔬 4 Experiments You Did Today!
A
Speed Racer
Change both delays to delay(100) and upload.
The LED blinks super fast — 10 times per second! Smaller delay = faster blink.
B
SOS Morse Code
Use delay(200) short and delay(600) long blinks for S-O-S pattern.
You sent a real distress signal! Sailors use S-O-S to say HELP ME.
C
Wand Check — Pin Voltage
Set multimeter to DC Voltage. Red probe to Pin 7, black to GND while blinking.
Display jumps: ~5.0V when HIGH, ~0.0V when LOW. Code and electricity match!
D
LED Voltage Measurement
Touch red probe to LED long leg (+) and black probe to LED short leg (−).
Reads ~1.8V–2.2V when ON. The resistor takes the rest of the 5V as heat!

🔧 Troubleshooting Quick Guide
ProblemQuick Fix
LED won't light upFlip LED — long leg (+) to resistor, short leg (−) to GND
LED very dimUse pin 7 (not analog) and 330Ω resistor (not 10kΩ)
Code won't uploadCheck Board and Port settings in Tools menu
LED stays always ONMake sure loop() has both HIGH and LOW with delays
Wand reads 0V on pin 7Check wire from pin 7 to breadboard is firmly inserted
