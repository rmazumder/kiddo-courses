📚 Key Words — Learn These!
📦 [ ]
Variable
A labeled box in the Arduino's memory that stores a value. The label stays the same, but the value inside can change anytime!
💬 >_
Serial Monitor
A secret window in the Arduino IDE where your Arduino can print messages. It gives your Arduino a voice to talk to you!
🔢 9600
Baud Rate
The speed of communication between the Arduino and your computer. Both must match — we use 9600 baud.
📊 ++
Counter
A variable that adds 1 to itself each loop. Like counting blinks: count = count + 1 keeps track!
📦 What is a Variable? The Labeled Box Idea
Imagine you have a box with a label on it. Inside you can store a number, a word, or a true/false answer. You can look inside, change what is in there, or use it in your code. That is a variable!
blinkSpeed
500
int — whole number
blinkCount
0
int — starts at zero
voltage
3.14
float — decimal number
myName
"Arduino"
String — text
Each variable has a TYPE (what kind of value), a NAME (the label), and a VALUE (what is inside the box).
int
Whole numbers only — no decimal point
int age = 10;
float
Decimal numbers — use for measurements
float temp = 23.5;
String
Words & text — always in quotes
String name = "Alex";
bool
Only true or false — yes or no
bool ledOn = true;
💬 The Serial Monitor — Giving Arduino a Voice!
Add two magic commands to make your Arduino talk:
1. Serial.begin(9600) in setup()
2. Serial.println("Hello!") to print
3. Click the magnifying glass icon (top-right of IDE) to open the window.
void setup() { Serial.begin(9600); Serial.println("Hello, World!"); Serial.println("I am Arduino!"); Serial.println("I am alive!"); } void loop() { // Nothing here yet }
What You See in the Serial Monitor:
Hello, World! I am Arduino! I am alive!
Serial.print() — prints on the same line.
Serial.println() — prints and then moves to a new line.
You can print text, numbers, or variables!

💡 My Activity Today — Build a Blink Counter!
Wire your LED circuit (pin 7, 330Ω resistor, GND), then type and upload this sketch:
int blinkSpeed = 500; // milliseconds int blinkCount = 0; // counter starts at 0 int ledPin = 7; void setup() { pinMode(ledPin, OUTPUT); Serial.begin(9600); Serial.println("=== Blink Counter Started! ==="); } void loop() { digitalWrite(ledPin, HIGH); delay(blinkSpeed); digitalWrite(ledPin, LOW); delay(blinkSpeed); blinkCount = blinkCount + 1; // add 1 each blink Serial.print("Blink #"); Serial.print(blinkCount); Serial.print(" | Speed: "); Serial.print(blinkSpeed); Serial.println(" ms"); }
The variable blinkCount goes up by 1 every blink — your Arduino is counting and reporting in real time!
✅ Steps to Complete:
  1. 1Wire LED to pin 7 with a 330Ω resistor to GND.
  2. 2Type the sketch above in the Arduino IDE and upload it.
  3. 3Click the magnifying glass to open the Serial Monitor — watch it count!
  4. 4Change blinkSpeed to 100 — watch the counter race!
  5. 5Add blinkSpeed = blinkSpeed + 50; inside loop() — watch it slow down!
