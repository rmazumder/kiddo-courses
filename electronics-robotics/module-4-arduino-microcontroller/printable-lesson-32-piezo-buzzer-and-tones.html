📚 Key Words — Learn These!
🔊 Hz
Frequency (Hz)
How many times per second something vibrates. Measured in Hertz (Hz). More vibrations per second = higher pitch sound!
🎶 ~
Passive Buzzer
A buzzer that needs a signal from the Arduino to make sound. We control the pitch with tone(). Not the same as an active buzzer!
🎹 ♫
tone() Function
The Arduino command that makes a buzzer vibrate at a chosen frequency. Format: tone(pin, Hz, ms). Use noTone() to stop.
🎵 8va
Octave
When you double a frequency you go up one octave. C4 = 262 Hz, C5 = 524 Hz — same note letter, but one step higher in pitch!
🔊 How Sound Works — It's All About Vibration!
Put your hand on your throat and HUM. Feel that vibration? That is exactly how a buzzer works! Your Throat (Humming) Piezo Buzzer on Arduino +----------------------+ +------------------------+ | Vocal cords vibrate | | Tiny disc vibrates | | FAST --> HIGH pitch | | FAST Hz --> HIGH sound | | SLOW --> LOW pitch | | SLOW Hz --> LOW sound | +----------------------+ +------------------------+ | | v v Air molecules shake Air molecules shake Ears hear SOUND! Speaker hears SOUND! Arduino pin 8 switches: 5V --- 0V --- 5V --- 0V ... At 440 Hz: switches 440 times per second = note A!
Higher frequency = faster vibration = higher pitch. Lower frequency = slower vibration = lower pitch.
⚖ Wire the Buzzer + The tone() Command
🔋 Buzzer Wiring (Step 2)
Arduino Uno Breadboard +-----------+ | Pin 8 |--[wire]--> (+) longer leg | | | GND |--[wire]--> (-) shorter leg +-----------+ Buzzer legs: (+) longer leg / marked with + (-) shorter leg Use a PASSIVE buzzer! Active buzzers have a fixed pitch.
Passive buzzer = needs Arduino signal to vibrate.
Active buzzer = has its own oscillator, pitch is fixed.
We want passive so we control the pitch!
★ Pin 8 (+) and GND (-)
🎹 The tone() Command (Step 3)
// Play a tone: tone(pin, frequency, duration); pin = buzzer pin (8) frequency = Hz value (440) duration = ms length (1000) // Stop a tone: noTone(pin); // Example -- note A for 1 second: tone(8, 440, 1000); delay(1050); noTone(8);
440 Hz = note A4 — the pitch orchestras use to tune!
Add a small delay() after each note so they don't blend together.
♫ tone(8, 440, 1000)
🎶 Musical Note Frequencies — 4th Octave
C4 = 262 Hz C#4 = 277 Hz D4 = 294 Hz D#4 = 311 Hz E4 = 330 Hz F4 = 349 Hz F#4 = 370 Hz
G4 = 392 Hz G#4 = 415 Hz A4 = 440 Hz A#4 = 466 Hz B4 = 494 Hz C5 = 523 Hz (one octave up!)
Higher numbers = higher pitch. C5 (523) is exactly double C4 (262) — that is what "one octave" means!

💡 My Activity Today — Build a Melody Machine!
// Twinkle Twinkle Little Star -- Full Melody Code (Step 5) int buzzerPin = 8; int C4=262, D4=294, E4=330, F4=349, G4=392, A4=440; int melody[] = {C4,C4,G4,G4,A4,A4,G4, F4,F4,E4,E4,D4,D4,C4}; int noteDurations[] = {400,400,400,400,400,400,800, 400,400,400,400,400,400,800}; int numNotes = 14; void setup() { for (int i = 0; i < numNotes; i++) { tone(buzzerPin, melody[i], noteDurations[i]); delay(noteDurations[i] + 50); // small gap between notes noTone(buzzerPin); } } void loop() { /* plays once in setup */ }
The melody[] array holds the notes, noteDurations[] holds how long each plays, and the for loop steps through them one by one. Arrays + loops = a real song!
