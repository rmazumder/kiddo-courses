📚 Key Words — Learn These!
🔄 {}
For Loop
A block of code that repeats a set number of times. Like climbing stairs — start at the bottom, go one step at a time, stop when you reach the top!
📁 []
Array
A list of values stored in one variable. Instead of one labeled box, an array is a whole row of labeled boxes — perfect for storing 6 LED pin numbers!
🔢 i
Index (i)
The position number inside an array. Arrays always start at index 0 — so the first item is [0], the second is [1], and so on.
▶ ++
i++ (Increment)
Add 1 to the variable i after each loop pass. i++ is shorthand for i = i + 1. It moves the loop counter to the next step!
🔄 How a For Loop Works — Anatomy of a Loop
The three parts inside for( ... ; ... ; ... )
int i = 0 START — set the counter to 0
i < 6 CONDITION — keep going while true
i++ STEP — add 1 each time
// This loop runs 6 times (i = 0, 1, 2, 3, 4, 5) for (int i = 0; i < 6; i++) { digitalWrite(leds[i], HIGH); // turn on LED at position i delay(100); digitalWrite(leds[i], LOW); // turn it off }
Each pass through the loop, i grows by 1: 0 → 1 → 2 → 3 → 4 → 5 → STOP. That is 6 LEDs controlled in just 4 lines of code!

💡 My Activity Today — Knight Rider LED Chaser!
Arduino Pins Breadboard +-----------+ | Pin 3 (~)|----[330 ohm]----LED 1----GND | Pin 5 (~)|----[330 ohm]----LED 2----GND | Pin 6 (~)|----[330 ohm]----LED 3----GND | Pin 9 (~)|----[330 ohm]----LED 4----GND | Pin 10 (~)|----[330 ohm]----LED 5----GND | Pin 11 (~)|----[330 ohm]----LED 6----GND | GND |----[common GND rail] +-----------+ GND rail: ================================ LED order: 1 2 3 4 5 6
Steps to Build
  • 1Place 6 LEDs in a row on the breadboard (leave 5 rows between each one)
  • 2Connect a 330-ohm resistor from each Arduino pin (3, 5, 6, 9, 10, 11) to each LED's long leg (+)
  • 3Connect all LED short legs (−) to the GND rail on the breadboard
  • 4Connect the GND rail to Arduino GND with a jumper wire
  • 5Upload the Knight Rider code and watch the light scan back and forth!
Use a for loop in setup() to call pinMode() on all 6 LEDs at once — no repeated code!
✅ Components I Used:
  • Arduino Uno
  • Breadboard
  • 6 LEDs
  • 6 x 330Ω resistors
  • 8 jumper wires
  • USB cable

🔬 4 Challenges You Tried Today!
A
Speed Control
Change delayTime to 50.
The scan becomes super fast — lower delay = faster light!
B
All On, All Off
Loop to turn ALL LEDs HIGH, then all LOW with delay(500).
One loop controls every LED at once — 6 lines instead of 12!
C
Stack Effect
Turn LEDs on one by one and leave them on, then turn off in reverse.
A second reverse loop (i--) can undo the forward loop!
D
Wand Check (Multimeter)
Upload an "all on" sketch and measure each pin to GND.
Every pin should read ~5V — if one reads 0V, check that LED's wiring!
