📚 Key Words — Learn These!
🔅 ~
PWM (Pulse Width Modulation)
The Arduino's trick of switching a pin ON and OFF hundreds of times per second so fast your eyes see it as dimmer. Like a super-fast light switch!
🔀 %
Duty Cycle
How much time the pin spends ON vs OFF in each blink. 50% duty cycle = on half the time = half brightness. More ON time = brighter!
🖍 ~
analogWrite()
The Arduino command that sets PWM brightness. Range: 0 (always off) to 255 (always on). Only works on pins marked with ~!
🔳 V
Average Voltage
PWM never truly outputs 2.5V — it flips between 0V and 5V. Your multimeter averages the pulses. analogWrite(127) → about 2.5V average!
⚡ The PWM Trick — How One Pin Fakes a Dimmer!
The LED is never truly dim — it blinks 490 times per second. Your eyes blur the flashes together!
Full brightness (255): [ON ][ON ][ON ][ON ][ON ][ON ][ON ][ON ] = 100% ON Half brightness (127): [ON ][OFF][ON ][OFF][ON ][OFF][ON ][OFF] = 50% ON Quarter bright ( 64): [ON ][OFF][OFF][OFF][ON ][OFF][OFF][OFF] = 25% ON Off ( 0): [OFF][OFF][OFF][OFF][OFF][OFF][OFF][OFF] = 0% ON analogWrite(pin, 127): 5V__ __ __ __ __ __ | | | | | | | | | | | 0V |__| |__| |__| |__| |__| | <------490x per second------> Wand reads: ~2.5V average
Average voltage = (analogWrite value ÷ 255) × 5V  |  Example: 127 ÷ 255 × 5V = 2.49V
💻 PWM Pins and the Fade Code
🆕 PWM Pins on Arduino Uno
Arduino Uno board: ~3 ~5 ~6 ~9 ~10 ~11 Look for the ~ (tilde) symbol! Only these 6 pins can use analogWrite().
Today we use pin ~9. Move your LED wire from pin 7 to pin 9 to unlock fade control!
The other digital pins can only do HIGH or LOW — no dimming possible.
💡 The Smooth Fade Loop
int ledPin = 9; void loop() { // Fade IN: dark to bright for (int b=0; b<=255; b++) { analogWrite(ledPin, b); delay(10); } // Fade OUT: bright to dark for (int b=255; b>=0; b--) { analogWrite(ledPin, b); delay(10); } }
The for loop counts brightness from 0 to 255 one step at a time — like a counter that adds 1 each round until it reaches the goal!

🔧 Troubleshooting Quick Guide
ProblemQuick Fix
LED stays on/off, no fadeCheck LED is on pin 9 (~), not pin 7
Fade is jerkyReduce delay to 5ms in the for loop
Pot dimmer not workingCheck pot middle pin goes to A0
Wand shows jumping numbersNormal! PWM is fast. Let Wand settle
analogWrite value wrongDivide analogRead by 4: 1023/4 = 255
