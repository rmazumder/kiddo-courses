📚 Key Words — Learn These!
🔍 IN
Digital Input
A pin that listens to the world. The Arduino reads it as either HIGH (5V) or LOW (0V) — like asking "is it on or off?"
📋 PU
INPUT_PULLUP
A built-in resistor that pulls a pin UP to 5V when nothing is connected. Press the button and it pulls the pin DOWN to 0V.
🮲 if
if / else
The Arduino's way of making decisions. "IF this is true, do THIS. ELSE do THAT." — a fork in the road for your code!
🔃 ==
digitalRead()
A command that checks a pin and returns HIGH or LOW. Use two equal signs == to compare the result in an if statement.
⚡ The Pull-Up Trick — Why Pressed = LOW
The Problem: A floating pin reads random garbage!
When a button is not pressed, the pin is disconnected from everything. It drifts between HIGH and LOW randomly. The fix? INPUT_PULLUP — a built-in resistor that gently holds the pin at 5V until you press the button and pull it down to GND.
NOT PRESSED: PRESSED: Pin 2 ---[internal pullup]---5V Pin 2 ---[internal pullup]---5V \ \ (button open) [BUTTON closed] | reads: HIGH (5V) GND reads: LOW (0V) REMEMBER: pressed = LOW | not pressed = HIGH (it seems backwards!)
Use pinMode(buttonPin, INPUT_PULLUP) in setup(). Then check: if (buttonState == LOW) → button is PRESSED!
💻 The if/else Statement — Making Decisions
How the Arduino decides what to do:
// Lesson 28: Button Controls LED int buttonPin = 2; int ledPin = 7; void setup() { pinMode(ledPin, OUTPUT); pinMode(buttonPin, INPUT_PULLUP); // built-in pull-up Serial.begin(9600); } void loop() { int buttonState = digitalRead(buttonPin); // HIGH or LOW if (buttonState == LOW) { // LOW = pressed! digitalWrite(ledPin, HIGH); // LED ON Serial.println("Button PRESSED -- LED ON"); } else { digitalWrite(ledPin, LOW); // LED OFF Serial.println("Button RELEASED -- LED OFF"); } }
Key detail: == (two equals) means "is equal to?" — one equals = sets a value, two equals = compares!

💡 My Activity Today — Build & Code!
Arduino Uno Breadboard +---------+ | Pin 2 |----[wire]----+--- Button (one side) | | | | GND |----[wire]----+--- Button (other side) | | | Pin 7 |----[wire]---[330 ohm]---LED(+)---LED(-)---GND +---------+ Button detail: 4 legs in a square -- straddle the center gap Pin 2 ---- one side | GND ---- other side
The button connects Pin 2 to GND when you press it. INPUT_PULLUP handles the rest — no extra resistor needed for the button!
Wiring Steps:
  • 1Place the push button across the center gap of the breadboard.
  • 2Connect one side of the button to Arduino pin 2 with a jumper wire.
  • 3Connect the other side of the button to Arduino GND.
  • 4Keep your LED from last lesson: pin 7 → 330Ω resistor → LED (+) → LED (−) → GND.
  • 5Upload the code, open Serial Monitor, and press the button!

ProblemQuick Fix

Button does nothingCheck pin 2 wire; confirm both button sides are connected
Serial shows random HIGH/LOWMust use INPUT_PULLUP, not just INPUT
