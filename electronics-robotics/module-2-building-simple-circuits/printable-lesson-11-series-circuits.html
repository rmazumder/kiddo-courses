📚 Key Words — Learn These!
🚊 V
Series Circuit
A circuit where components are connected one after another on a single path — like train cars on a single track. One break stops everything!
⚡ I
Current (Same Everywhere)
In a series circuit, the same amount of current flows through every component. It never splits — there is only one track!
📈 V
Voltage Drop
Each component "uses up" some of the battery's push. The voltage drops a little at every stop along the series track.
➕ ∑
Kirchhoff's Voltage Law
All the voltage drops in a series circuit add up to the battery voltage. Every volt is accounted for — nothing disappears!

📋 4 Golden Rules of Series Circuits
🚀
One Path Only
Electricity has just ONE route to travel — no shortcuts or side roads. Every component is on the same single track.
⚡
Same Current Everywhere
The current (amps) is identical at every point in the circuit. Like the same train passing every station.
📈
Voltage Gets Split
Each component uses some voltage. Bigger resistance = bigger voltage drop. The battery's push is shared across all components.
🚫
Remove One = All Stop
Break the chain anywhere and current stops everywhere. Like old holiday lights — one broken bulb kills the whole string!

🔬 4 Experiments You Did Today!
A
The Voltage Treasure Hunt
Measure voltage across R1, R2, and the battery.
V_R1 + V_R2 = battery voltage — every volt is accounted for!
B
Three LEDs in Series
Wire red, green, and yellow LEDs one after another.
They light up but dimmer — each LED uses ~2V, sharing the 9V battery!
C
The Break Test
Remove the green LED from the series chain.
All LEDs go dark instantly — just like old holiday lights!
D
Series Resistance Check
Measure resistance across R1 + R2 together with the Wand.
Wand reads ~800Ω — exactly 330 + 470 added together!
