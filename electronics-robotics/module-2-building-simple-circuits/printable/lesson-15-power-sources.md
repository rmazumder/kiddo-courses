## Test Any Battery with Your Wand

**How to measure battery voltage:**

1. Set Wand to **DC Volts, 20V range**
2. RED probe → + terminal (small bump on battery)
3. BLACK probe → − terminal (flat end)
4. Read display!

**Is it fresh or dying?**

| Battery | Fresh (Good) | Getting Tired | Dead |
|---------|------------|---------------|------|
| **9V** | 9.0–9.6V | 7.0–8.9V | <7V |
| **AA/AAA** | 1.5–1.6V | 1.2–1.4V | <1.1V |
| **CR2032** | 3.0–3.3V | 2.5–2.9V | <2.5V |

✓ **Your Wand = instant battery tester!**

## Voltage vs. Capacity

Two different things:

| Concept | Meaning | Example |
|---------|---------|---------|
| **Voltage** | How HARD it pushes (force) | 9V pushes harder than 1.5V |
| **Capacity (mAh)** | How LONG it lasts (fuel tank) | AA lasts longer than 9V |

**Analogy:** 9V = sports car (powerful, runs out fast)
AA = minivan (steady, lasts longer)

| Battery | Voltage | Capacity | Runtime |
|---------|---------|----------|---------|
| 9V | 9V | ~500 mAh | Sprinter ⚡ |
| AA | 1.5V | ~2500 mAh | Marathon runner 🏃 |
| AAA | 1.5V | ~1000 mAh | Jogger |
| CR2032 | 3V | ~225 mAh | Short walk |

## Batteries in SERIES: Stack Voltage!

**Problem:** One AA (1.5V) is not enough for some projects.
**Solution:** Connect batteries end-to-end in SERIES.

```
[AA1: 1.5V] → [AA2: 1.5V] → [AA3: 1.5V] → [AA4: 1.5V]
+    −          +    −          +    −          +    −

Total = 1.5 + 1.5 + 1.5 + 1.5 = 6V!
```

✓ **Voltages STACK**

**Real-world example:** 4-AA battery holder = 6V (this is how flashlights work!)

**Wand Check:**

| Config | Predicted | Wand Reads |
|--------|-----------|-----------|
| 1 AA | 1.5V | _____ |
| 2 AA series | 3.0V | _____ |
| 3 AA series | 4.5V | _____ |
| 4 AA series | 6.0V | _____ |

## Batteries in PARALLEL: More Battery Life!

**Problem:** You need 1.5V but want it to last 3X longer.
**Solution:** Connect batteries side-by-side in PARALLEL.

```
[AA1: 1.5V] --+-- (+) Output: 1.5V
[AA2: 1.5V] --+     BUT capacity = 3X longer!
[AA3: 1.5V] --+
              |
              +-- (−)
```

✓ **Voltage stays SAME, capacity TRIPLES**

**Key rule:** Only parallel batteries of the SAME type and age!
(Old + new in parallel = new tries to charge old = heat = danger!)

**Wand Check:**

| Config | Predicted | Wand Reads |
|--------|-----------|-----------|
| 1 AA | 1.5V | _____ |
| 2 AA parallel | 1.5V (same!) | _____ |

✓ **Voltage unchanged, but last 2X longer!**

## Inside a 9V Battery (The Secret!)

**Question:** What's INSIDE a 9V battery?

**Answer:** Six tiny AA cells wired IN SERIES!

```
+-------+
|  ▀▀▀  |  ← 6 tiny cells
| 1.5V  |  ← each is 1.5V
| each  |  ← 6 × 1.5 = 9V!
+-------+
```

✓ **9V = secret stack of 6 tiny 1.5V batteries!**

## Battery Safety Rules

```
BATTERY SAFETY RULES
====================

1. ✓ Never short-circuit (+ to − directly)
   → Gets HOT, can leak or burst

2. ✓ Never throw batteries in fire
   → Can EXPLODE

3. ✓ Always check polarity before inserting
   → Wrong way = won't work, may damage circuit

4. ✓ Remove unused batteries from devices
   → Prevent leaks

5. ✓ Don't mix old + new in series
   → Old one works too hard, leaks

6. ✓ Dispose properly
   → Many stores have battery recycling

7. ✓ HOT/SMELLY/SWOLLEN battery?
   → STOP! Tell an adult immediately!
```

## Key Takeaways

1. **Different batteries = different voltages** → 9V ≠ AA (bigger ≠ stronger)
2. **Use Wand to test** → Find dead batteries in seconds
3. **SERIES = stack voltage** → More PUSH (use when you need more voltage)
4. **PARALLEL = extend runtime** → More CAPACITY (use when you need it to last)
5. **Safety first** → Respect the chemistry inside!
