📚 Key Words — Learn These!
⏰ 555
555 Timer
An IC that generates a clock pulse — a rapid ON/OFF signal. In fast mode it pulses thousands of times per second, spinning the counter too fast to see!
🔄 4017
4017 Decade Counter
An IC that counts pulses. Each pulse moves to the next output (Q0 → Q1 → Q2…). We limit it to 6 outputs to get numbers 1–6 like a real die!
🔌 →|
Diode
A one-way traffic gate for electricity — current can only flow in the direction of the stripe. We use them so multiple 4017 outputs can share the same LED without interfering.
🎲 ?
Electronic Random
The 555 runs so blindingly fast that when you release the button, the counter stops on a number that no human could predict. That unpredictability IS the randomness!
🎲 How the Digital Dice Works
System Block Diagram — Signal Flow
Button Press (hold) | v +-------------+ clock pulses +----------------+ which LEDs? +-----------+ | 555 Timer | ------------------> | 4017 Counter | -----------------> | 7 LEDs | | (fast clock)| ~6,857 Hz! | (cycles Q0-Q5) | diode matrix | (dice face| | ~6,857 Hz | | resets at Q6 | | pattern) | +-------------+ +----------------+ +-----------+ ^ | Button RELEASE --> 555 stops --> counter FREEZES --> random number shown!
Hold the button = counter spins invisible fast. Release = random number freezes on the LED dice face!
🔌 Dice Face Layout & LED Assignments
■ LED Position Map (7 LEDs)
+-----+-----+-----+ | A | | B | +-----+-----+-----+ | C | G | D | +-----+-----+-----+ | E | | F | +-----+-----+-----+ G = center dot A,B = top corners C,D = middle sides E,F = bottom corners
G (center) is shared by numbers 1, 3, and 5.
Use diodes so multiple outputs can light G at the same time!

# | LEDs On | 4017 Pin --|---------------|---------- 1 | G | Q0 Pin 3 2 | B, E | Q1 Pin 2 3 | B, G, E | Q2 Pin 4 4 | A, B, E, F | Q3 Pin 7 5 | A,B,G,E,F | Q4 Pin 10 6 | A,B,C,D,E,F | Q5 Pin 1 Reset trick: Q6 (Pin 5) --> Reset (Pin 15) = counter loops 0-5 only!

🔬 4 Key Build Challenges Today!
A
Speed Up the 555
Use C1 = 0.01μF (tiny!) and R1=1k, R2=10k in your 555 astable circuit.
Formula: 1.44 ÷ ((1k+20k) x 0.01μF) = ~6,857 Hz — too fast to see! The Wand reads ~4–5V (averaging rapid pulses).
B
Limit the Counter to 6
Wire 4017 Pin 5 (Q6) directly to Pin 15 (Reset) with a jumper wire.
The counter now resets immediately when it reaches Q6 — giving exactly 6 counting positions for a real die face!
C
Diode Matrix for Shared LEDs
Wire LED G (center dot): connect Q0, Q2, and Q4 each through a 1N4148 diode, join together, then through 330Ω to LED G.
The diode stripe (cathode) points AWAY from the 4017 output. Without diodes, outputs would interfere with each other!
D
Wand Debug Slow-Mode Test
Temporarily swap C1 for a 47μF cap to slow the 555 way down. Watch the 4017 outputs one by one with your Wand.
Only ONE output should be HIGH at any moment. Verify all 6 dice patterns light the correct LEDs before speeding back up!

🔧 Troubleshooting Quick Guide
ProblemQuick Fix
Always same number555 not running — check button Pin 4 connection
Wrong LED patternsCheck diode directions — stripe = cathode = away from 4017
Counter goes past 6Verify Q6 (Pin 5) wired to Reset (Pin 15)
Some LEDs don't lightCheck LED polarity + 330Ω resistor in place
All LEDs on at onceDiodes backwards — flip each one
Battery dies quicklyMeasure with Wand — replace if below 7V
