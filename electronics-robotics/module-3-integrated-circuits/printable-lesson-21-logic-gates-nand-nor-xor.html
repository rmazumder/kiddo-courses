📚 Key Words — Learn These!
🔪 ¬
NAND Gate
NOT-AND — flips the AND result. Output is 0 only when both inputs are 1. Called the "Universal Gate" — it can pretend to be any other gate!
🚫 ↓
NOR Gate
NOT-OR — flips the OR result. Output is 1 only when both inputs are 0. If anything is ON, the output goes OFF.
🔎 ≠
XOR Gate
Exclusive OR — the "disagree detector." Output is 1 only when the inputs are different. Same inputs = always 0.
🌏 U
Universal Gate
A gate that can build every other gate by itself. NAND is universal — tie both inputs together and you get a NOT gate for free!

⚡ NAND's Superpower & XOR's Secret
🔪 NAND = The Universal Gate
Tie both inputs together: Input --+-- Pin 1 | +-- Pin 2 | Pin 3 = Output Input 0 → Output 1 (it's a NOT!) Input 1 → Output 0 (it's a NOT!)
Connect both inputs of a NAND gate together and it becomes a NOT gate! Engineers love NAND because one chip can build an entire computer.
IC chip: 7400 (4 gates inside)
⚡ Pins: In=1,2   Out=3   VCC=14   GND=7
🔎 XOR = The Hallway Light Switch
Two switches, one light: Switch A Switch B Light Same pos Same pos OFF Different positions ON "I only light up when we DISAGREE with each other!"
Think of a hallway with a switch at each end. When both are up or both are down — light OFF. When they disagree — light ON!
Real use: Adding binary numbers in computers!
⚡ Pins: In=1,2   Out=3   VCC=14   GND=7

💡 My Activity Today — Build All Three Gates!
NAND (7400): Pin1+Pin2 = Inputs → Pin3 = Output | VCC=Pin14 GND=Pin7 NOR (7402): Pin2+Pin3 = Inputs → Pin1 = Output | VCC=Pin14 GND=Pin7 ← different! XOR (7486): Pin1+Pin2 = Inputs → Pin3 = Output | VCC=Pin14 GND=Pin7 Each gate wired the same way: Button A ──[10kΩ pull-down]──┐ ├── IC Input A ──┐ │ ├──[IC Gate]── Pin OUT ──[330Ω]── LED(+) ── GND Button B ──[10kΩ pull-down]──┘── IC Input B ──┘
The 7402 NOR IC is the sneaky one — its output pin comes FIRST (Pin 1), not last!
  1. 1Place the 7400 IC on the breadboard. Connect Pin 14 to (+) rail and Pin 7 to (−) rail.
  2. 2Add two push-buttons with 10k pull-down resistors. Connect to Pin 1 and Pin 2.
  3. 3Connect Pin 3 (output) through a 330Ω resistor to an LED, then LED to GND.
  4. 4Test all 4 button combos. Both ON = LED OFF! (Opposite of AND.)
  5. 5Repeat with 7402 (NOR) and 7486 (XOR) ICs. Note: 7402 output is Pin 1!
  6. 6Use your Magic Wand (multimeter, DC Voltage) to measure output. HIGH ≈ VCC, LOW ≈ 0V.

🔬 4 Experiments You Did Today!
A
NAND Gate Test
Press both buttons at the same time on the NAND circuit.
LED turns OFF — the only time NAND says no! Every other combo = LED ON.
B
NAND Becomes NOT
Connect Pin 1 and Pin 2 together with a wire, then use one button.
One NAND gate turns into a NOT gate! Press = OFF, Release = ON.
C
NOR Security Alarm
Build the NOR circuit. Press either button (or both).
LED stays ON only when BOTH inputs are 0 — like an alarm in safe mode!
D
XOR Disagree Detector
Build the XOR circuit. Try same buttons vs. different buttons.
LED only lights when the buttons DISAGREE — the ultimate difference detector!
