📚 Key Words — Learn These!
🌡 S
Sensor
A device that detects something in the real world (heat, light, movement) and converts it into an electrical signal your Arduino can read.
💥 D
DHT11
A digital temperature and humidity sensor. Measures 0–50°C and 20–90% humidity. Sends its data as a digital signal.
💧 H
Humidity
How much invisible water vapor is in the air. 100% = rainforest-muggy. 20% = desert-dry. Your breath raises it!
📄 R
Pull-up Resistor
A resistor that keeps a signal wire at a steady HIGH voltage when the sensor is not talking — stops the wire from floating around like a balloon!
🔌 DHT11 Sensor — Pinout & Wiring Diagram
DHT11 MODULE (front view) ARDUINO WIRING +-------+-------+-------+ | SIG | VCC | GND | Arduino 5V --------> DHT11 VCC (middle) | (data)| | | Arduino GND --------> DHT11 GND (right) +-------+-------+-------+ Arduino Pin 2 ------> DHT11 SIG (left) | | | | | +---- BLACK wire -> GND rail | +------------ RED wire -> 5V rail +-------------------- DATA wire -> Pin 2 Pull-up resistor (10kΩ): between SIG pin and 5V SIG pin ----[10kΩ]---- 5V (Many 3-pin PCB modules already have this built in!)
The 10kΩ pull-up keeps the signal line at HIGH when the sensor is idle — without it, Arduino gets confused!
💻 Your First Temperature Reading — The Code
#include <DHT.h> #define DHTPIN 2 // Data wire on pin 2 #define DHTTYPE DHT11 // We are using DHT11 DHT dht(DHTPIN, DHTTYPE); void setup() { Serial.begin(9600); dht.begin(); // Wake up the sensor } void loop() { delay(2000); // DHT11 reads once per 2 seconds float temperature = dht.readTemperature(); // Celsius float humidity = dht.readHumidity(); if (isnan(temperature) || isnan(humidity)) { Serial.println("ERROR: Could not read sensor!"); return; // isnan() = "Is Not A Number" check } Serial.print("Temp: "); Serial.print(temperature); Serial.print(" C "); Serial.print("Humid: "); Serial.print(humidity); Serial.println(" %"); }
Install the library first: Sketch > Include Library > Manage Libraries > search "DHT sensor library" by Adafruit > Install All
📊 DHT11 vs DHT22 — Which One Are You Using?
🌡 DHT11 — Starter Wand
Range: 0 to 50 degrees C Humidity: 20 to 90 % Accuracy: +/- 2 degrees C Speed: 1 reading / second
Great for beginners! Very cheap, works perfectly for classroom projects. Already on most starter kits.
Think of it as the "learner" thermometer.
🌛 Perfect for indoor rooms
🌟 DHT22 — Pro Upgrade
Range: -40 to 80 degrees C Humidity: 0 to 100 % Accuracy: +/- 0.5 degrees C Speed: 1 reading / 2 seconds
More accurate and wider range — measures freezing temperatures too! Costs a bit more but worth it for weather stations.
Same code, just change DHT11 to DHT22.
⛄ Can measure outdoor cold!

✅ Wiring Steps:
  • 1Connect DHT11 VCC to Arduino 5V (red wire)
  • 2Connect DHT11 GND to Arduino GND (black wire)
  • 3Connect DHT11 Signal to Arduino Pin 2 (yellow wire)
  • 4Add 10kΩ pull-up resistor between Signal and 5V (skip if module has it built in)
  • 5Upload sketch, open Serial Monitor — readings appear every 2 seconds!

🔧 Troubleshooting Quick Guide
ProblemQuick Fix
Serial shows "ERROR"Check data wire on pin 2; add pull-up resistor
Reads 0 or NaNInstall both DHT + Adafruit Unified Sensor libraries
LCD shows nothingTry address 0x3F instead of 0x27 in the code
Temperature wrongDHT11 is ±2°C accuracy — that is normal!
Wand reads 0V on VCCTrace 5V wire from Arduino to sensor — loose connection
Updates very slowlyDHT11 max speed is 1 reading/sec — 2-sec delay is correct
