📚 Key Words — Learn These!
🤖 SRV
Servo Motor
A motor with built-in position control. Tell it an angle (0–180°) and it moves exactly there — and holds! Like a robot arm joint.
🔌 PWM
PWM Signal
Pulse Width Modulation — tiny repeating electrical pulses sent down the signal wire. Short pulse = 0°. Long pulse = 180°.
🎲 POT
Potentiometer
A dial (knob) that outputs variable voltage 0–5V. Turn it left or right to manually control the servo angle. Reads as 0–1023 on Arduino.
⚙️ MAP
map() Function
An Arduino command that converts one number range to another. Converts potentiometer values (0–1023) into servo angles (0–180°).
⚙️ How a Servo Works — The Clock Hand Analogy
Inside an SG90 Servo: +---------------------------+ | [Gear Train] | | Small DC motor | | Position sensor (pot) | | Control circuit board | +---------------------------+ | | | Brown Red Orange (GND) (VCC) (Signal)
Servo sweeps 0° → 180°
like a clock hand — and holds position!
Think of it like a clock hand: point to 3 o'clock (90°) or 6 o'clock (180°) — and it stays there until told otherwise!
🔌 Three Wires & First Sweep Code
Wire Color Connects To
Ground Brown / Black Arduino GND
Power Red Arduino 5V
Signal Orange / Yellow Pin 9 (PWM)
⚠ Only one SG90 on Arduino 5V.
Multiple servos need external power!
#include <Servo.h> Servo myServo; void setup() { myServo.attach(9); Serial.begin(9600); } void loop() { // Sweep 0 -> 180 for (int a = 0; a <= 180; a += 5) { myServo.write(a); delay(50); } // Sweep 180 -> 0 for (int a = 180; a >= 0; a -= 5) { myServo.write(a); delay(50); } }
🎲 Control With a Potentiometer Dial
🎲 Potentiometer Wiring
Potentiometer Arduino Left pin -------> GND Middle pin ------> A0 Right pin -------> 5V
The middle (wiper) pin outputs 0 to 5V as you turn the knob. Arduino reads it as 0 to 1023 via analogRead(). Three wires, just like the servo!
🎲 Dial left = 0°  |  Dial right = 180°
💻 Knob-to-Angle Code
int val = analogRead(A0); // 0 to 1023 int angle = map(val, 0, 1023, 0, 180); myServo.write(angle); delay(20);
map() converts the pot's 0–1023 range to 0–180 degrees. Turn the knob and the servo follows in real time — you are the controller!

🔧 Troubleshooting Quick Guide
ProblemQuick Fix
Servo does not moveSignal wire must be on PWM pin (3,5,6,9,10,11)
Servo jitters / vibratesAdd 100μF capacitor between 5V and GND near servo
Small range onlyUse Servo.h library and write() with 0–180
Pot control jumpyAdd delay(15) in loop — servo needs time to move
Arduino resetsServo draws too much current — use external 5V supply
Grinding noiseHitting mechanical limit — stay within 0–180°
