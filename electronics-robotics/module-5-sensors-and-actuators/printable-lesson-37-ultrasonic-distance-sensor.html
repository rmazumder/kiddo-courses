📚 Key Words — Learn These!
🔈 ))))
Echolocation
Using sound echoes to find distance. Bats do it in the dark! The HC-SR04 does the same thing — send a sound, listen for the bounce.
🔌 ~
Ultrasonic
Sound waves too high-pitched for humans to hear — above 20,000 Hz. The HC-SR04 uses 40,000 Hz. Completely silent to your ears!
🖌 T
Trigger Pin
The "shout" pin. Arduino sends a 10-microsecond pulse here to tell the sensor to fire a burst of ultrasonic sound waves.
🔋 E
Echo Pin
The "listen" pin. Goes HIGH for exactly as long as it took the sound to travel to the object and back. Longer = farther away!
🔈 How It Works — The Bat Superpower!
HC-SR04 ECHOLOCATION — The 4-Step Sonar Ping Arduino HC-SR04 Sensor Wall / Object -------- ---------------- ------------- Pin 9 [TRIG] [ECHO] (Trigger) ---10us---> [ (o) (o) ] ----))))))))----> | | | | (Echo) <---pulse--- [ ] <----((((((((---- | | | | Pin 10 (echo bounces back) Step 1: Arduino sends a 10-microsecond HIGH pulse to TRIG pin Step 2: Sensor fires 8 ultrasonic pulses at 40,000 Hz (you can't hear this!) Step 3: Pulses bounce off the nearest object and return as an echo Step 4: ECHO pin stays HIGH for the round-trip travel time DISTANCE FORMULA: Speed of sound = 0.034 cm per microsecond Distance (cm) = echo_duration_us * 0.034 / 2 ^-- divide by 2: sound goes THERE and BACK
Just like a bat screaming in the dark — send a sound, time the echo, calculate the distance!
🔌 Wire It Up — Then Read the Distance!
🔌 HC-SR04 Pinout
HC-SR04 (front view) +------+------+------+------+ | VCC | Trig | Echo | GND | +------+------+------+------+ | | | | 5V Pin 9 Pin 10 GND HC-SR04 -----> Arduino VCC -----> 5V Trig -----> Pin 9 Echo -----> Pin 10 GND -----> GND
Range: 2 cm to 400 cm (1 inch to 13 feet!)
Accuracy: about +/- 3 mm
Power: needs 5V — not 3.3V!
Only 4 wires. Easy to connect!
💻 First Distance Sketch
int trigPin = 9; int echoPin = 10; void setup() { pinMode(trigPin, OUTPUT); pinMode(echoPin, INPUT); Serial.begin(9600); } void loop() { digitalWrite(trigPin, LOW); delayMicroseconds(2); digitalWrite(trigPin, HIGH); delayMicroseconds(10); digitalWrite(trigPin, LOW); long dur = pulseIn(echoPin, HIGH); int dist = dur * 0.034 / 2; Serial.print("Distance: "); Serial.print(dist); Serial.println(" cm"); delay(100); }
pulseIn() measures how many microseconds the echo pin stays HIGH.
Upload, open Serial Monitor at 9600 baud, and wave your hand!

💡 My Activity Today — Build a Parking Sensor!
PARKING SENSOR WIRING Arduino Pin 9 ---[wire]---> HC-SR04 Trig Arduino Pin 10 ---[wire]---> HC-SR04 Echo Arduino Pin 8 ---[wire]---> Buzzer (+) Arduino Pin 5 ---[330ohm]-> Green LED -> GND Arduino Pin 6 ---[330ohm]-> Yellow LED -> GND Arduino Pin 7 ---[330ohm]-> Red LED -> GND Arduino 5V ---[wire]---> HC-SR04 VCC Arduino GND ---[wire]---> HC-SR04 GND + Buzzer (-) + LED GND rail DISTANCE ZONES: dist > 50 cm --> Green LED on, no beep (all clear!) dist > 20 cm --> Yellow LED on, slow beep (caution) dist > 5 cm --> Red LED on, fast beep (warning!) dist <= 5 cm --> Red LED on, rapid beep (STOP!)
You just built the same system real cars use to help drivers park! Yours cost about $3 in parts.
Wiring Steps — Check Each One:
  • 1Connect HC-SR04: VCC to 5V, GND to GND, Trig to Pin 9, Echo to Pin 10
  • 2Connect buzzer: (+) to Pin 8, (−) to GND rail
  • 3Connect green LED + 330Ω resistor from Pin 5 to GND
  • 4Connect yellow LED + 330Ω resistor from Pin 6 to GND
  • 5Connect red LED + 330Ω resistor from Pin 7 to GND
  • 6Upload the parking sensor sketch and walk toward the sensor slowly!

🔬 4 Experiments You Did Today!
A
Check the Accuracy
Hold an object exactly 10 cm away. Check Serial Monitor.
Arduino should read 10 cm ± 1 cm. A $2 sensor accurate to 3 mm!
B
Test the Zones
Walk slowly toward the parking sensor from 1 meter away.
Green → yellow → red → rapid beeps — watch the LEDs change!
C
Wand Check — Echo Pin
Set Wand to DC Volts. Touch red probe to Echo pin (Pin 10).
Hand close = ~1-2V average. Hand far = ~0.1V. Distance encoded as voltage!
D
Soft vs Hard Surfaces
Point the sensor at a pillow, then a book at the same distance.
Hard flat surfaces reflect better. Soft or angled surfaces confuse the sensor!
