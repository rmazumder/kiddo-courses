📚 Key Words — Learn These!
👁 PIR
PIR (Passive Infrared)
Detects infrared heat from warm bodies. “Passive” means it never sends anything out — it only watches and receives.
🌟 IR
Fresnel Lens
The white plastic dome on top of the PIR. Splits its view into many zones — motion triggers when a warm body crosses from one zone to another.
🔔 HIGH
Output Pin (HIGH/LOW)
Goes HIGH (~3.3V) when motion is detected, LOW (0V) when nothing moves. A clean digital signal — no in-between!
⏱ ms
Time Delay (millis)
How long the light stays on after detecting motion. Adjusted by a tiny pot on the HC-SR501 module — from 3 seconds up to 5 minutes!
🔍 How the HC-SR501 PIR Module Works
HC-SR501 PIR SENSOR — BOTTOM VIEW DETECTION ZONES (top view) +—————————————————+ Fresnel Lens splits view: | | +——————————+ | [Sensitivity] [Time Delay] | | Z1 | Z2 | Z3 | Z4 | | left pot right pot | +——————————+ | 3m — 7m 3s — 300s | Warm body crosses Z2→Z3? | | => MOTION DETECTED! HIGH! | VCC OUT GND | | | | | | Still in same zone? +—————————————————+ => No trigger. LOW. | | | 5V pin 7 GND ←— connect to Arduino
The Fresnel lens is like a spy-movie laser grid — cross from one zone to another and the alarm fires!
⚙ Wire It Up + Your First Motion Detection Code
🔌 3-Wire Connection
PIR Sensor Arduino ——————— —————— VCC —————> 5V OUT —————> Pin 7 GND —————> GND
Just 3 wires — one of the simplest sensors to connect!
Important: After powering on, stay very still for 30–60 seconds while the PIR calibrates its baseline infrared level. Then it is ready!
💡 Smart Night Light Logic
PIR detects motion? YES LDR reads dark? YES —————————————— BOTH true? → LED ON! Only one true? → LED OFF
Combine the PIR with an LDR (light sensor) so the light only activates when there is motion AND it is dark. This is how real smart homes save electricity!
int pirPin = 7; int ledPin = 13; // PIR on pin 7, LED on pin 13 void setup() { pinMode(pirPin, INPUT); pinMode(ledPin, OUTPUT); Serial.begin(9600); Serial.println("Warming up 30 sec — stay still!"); delay(30000); Serial.println("PIR READY!"); // 30s calibration } void loop() { int motion = digitalRead(pirPin); // Read PIR: HIGH or LOW if (motion == HIGH) { digitalWrite(ledPin, HIGH); Serial.println("*** MOTION! ***"); } else { digitalWrite(ledPin, LOW); Serial.println("All clear..."); } delay(200); }

WIRING STEPS: 1. PIR VCC → Arduino 5V (red wire) 2. PIR GND → Arduino GND (black wire) 3. PIR OUT → Arduino Pin 7 (yellow wire) 4. LED (+) → 330Ω resistor → Arduino Pin 9 5. LED (-) → Arduino GND 6. Buzzer + → Arduino Pin 8 (optional alarm!) 7. Power on → stay still 30 sec for calibration 8. Wave your hand → LED lights up + "MOTION!" in Serial Monitor SECURITY LIGHT TIMER: change lightDuration = 10000 to adjust (ms) ALARM MODE: set alarmMode = true for buzzer alert

🔬 4 Experiments You Did Today!
A
Walk Across the Room
Walk slowly across the room at different distances from the sensor.
The PIR has a range of 3–7m — adjust the sensitivity pot to change how far it can see!
B
Hot Cup Test
Place a warm cup of water in front of the sensor. Then move it slowly past.
The PIR detects any warm object — not just people. Movement is what triggers it, not just heat!
C
Wand Check — HIGH vs LOW
Set Wand to DC Volts. Red probe to PIR OUT (pin 7), black to GND. Wave hand, then freeze.
Voltage snaps from 0V to 3.3V instantly — a clean digital signal unlike analog sensors!
D
Smart Night Light
Add the LDR. Cover the LDR with your hand (simulating dark) then wave in front of the PIR.
LED only fires when BOTH conditions are met — this is real smart-home logic at work!

🔧 Troubleshooting Quick Guide
ProblemQuick Fix
False triggers randomlyWait full 60s warm-up; keep away from heaters and air vents
No detection at allCheck wiring; confirm Fresnel dome is on top; turn sensitivity pot up
LED stays on foreverTime delay pot is at max — turn counter-clockwise for shorter delay
Night light never firesPrint lightLevel to Serial Monitor and adjust darkThreshold
Wand reads 5V not 3.3VSome modules output 5V — both values read as HIGH on Arduino
