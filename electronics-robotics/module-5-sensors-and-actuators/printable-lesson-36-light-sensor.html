📚 Key Words — Learn These!
🔍 LDR
LDR (Light Dependent Resistor)
A resistor whose resistance changes with light. Bright light = low resistance. Darkness = very high resistance. Also called a photoresistor!
⛭ ÷
Voltage Divider
Two resistors in series that split voltage so Arduino can read a changing signal. The LDR + 10kΩ resistor form one together.
📈 A0
Analog Input
Arduino pin A0 reads voltages from 0 to 5V and converts them to numbers 0–1023. Higher voltage = bigger number!
🔄 map()
map() Function
A code translator that rescales numbers from one range to another. Turns 0–1023 sensor values into 0–255 LED brightness — and can flip the relationship too!
⛭ The LDR Voltage Divider Circuit
Think of it like magic sunglasses: bright light = clear lenses (low resistance), darkness = black lenses (high resistance)!
Arduino 5V ----+ | [LDR] bright = ~200 ohms dark = ~500,000 ohms | +-------> Arduino A0 (reads voltage here) | [10k ohm] (fixed resistor - the "partner") | Arduino GND ---+ BRIGHT light: LDR resistance LOW --> voltage at A0 HIGH (~5V) --> reading ~1023 DARK room: LDR resistance HIGH --> voltage at A0 LOW (~0V) --> reading ~0
The LDR and the 10kΩ resistor share the 5V between them. When LDR resistance drops (bright), A0 climbs. When LDR resistance rises (dark), A0 falls. Arduino reads this changing voltage as a number!
💡 Light Levels & the map() Translator
🌞 LDR Resistance vs Light
Bright sun ... ~100-500 ohms (LOW) Room light .. ~1,000-5,000 ohms Dim room .... ~10,000-50,000 ohms Total dark .. ~200,000-500,000+ ohms (HIGH!)
Bright light knocks electrons loose → current flows easily → low resistance.
Darkness = very few free electrons → current blocked → huge resistance!
💡 Night Light Code (key lines)
int lightValue = analogRead(A0); // map() flips AND rescales: // dark (low reading) = bright LED // bright (high reading) = LED off int ledBrightness = map(lightValue, 0, 1023, 255, 0); if (lightValue < 300) { analogWrite(9, ledBrightness); } else { analogWrite(9, 0); // LED off }
map(value, fromLow, fromHigh, toLow, toHigh)
Rescales AND inverts in one line — dark room means bright LED automatically!

🔧 Troubleshooting Quick Guide
ProblemQuick Fix
Reading stuck at 0 or 1023Check voltage divider — LDR and 10kΩ must be in series between 5V and GND with A0 in the middle
LED doesn't respond to lightMake sure LED is on a PWM pin (3, 5, 6, 9, 10, or 11)
Readings jump wildlyAdd delay(100) and average 3 readings
Wand shows "OL" in darkSwitch to higher ohms range — LDR can be very high resistance
Night light flickers on/offAdjust threshold value in code to match your room's light level
LDR seems unresponsiveTurn the squiggly side to face the light source — the back is less sensitive
