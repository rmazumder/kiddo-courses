📚 Key Words — Learn These!
⚙ M
DC Motor
A motor that spins continuously when you give it voltage. Flip the voltage and it spins the other way — great for wheels and fans!
🔌 L
L298N Motor Driver
A powerful middleman chip that takes Arduino's tiny signals and delivers the big current a motor needs — up to 800 mA per channel!
🏛 H
H-Bridge
The circuit inside the L298N that looks like an "H". It can send current through the motor in either direction — that's how you reverse!
📈 ~
PWM Speed Control
Sending pulses of power on and off very fast. More "on" time = more speed. analogWrite(ENA, 128) = 50% speed!
💧 The Fire Hose Analogy — Why We Need a Motor Driver
Arduino Pin L298N Motor Driver DC Motor (garden hose: (fire hose pump: (fire hose load: max 40 mA) handles 800+ mA) wants 200-800 mA) [Arduino]--IN1/IN2-->[ L298N H-Bridge ]--OUT1/OUT2-->[Motor] ENA (PWM speed) | | 9V Battery (+) -------> +12V terminal (motor power) Battery GND --| Arduino GND --+--> GND terminal (SHARED GROUND!) KEY: Arduino whispers the instructions. L298N supplies the POWER to the motor.
Without the L298N, plugging a motor into Arduino is like connecting a fire hose to a garden tap — it would damage the Arduino!
⚛ L298N Module Layout — Know Your Pins!
🔎 Direction Control Truth Table — Motor A (IN1 + IN2)
IN1 Pin IN2 Pin Motor A Action Arduino Code
HIGH LOW Forward (spins one way) digitalWrite(IN1,HIGH); digitalWrite(IN2,LOW);
LOW HIGH Backward (other way) digitalWrite(IN1,LOW); digitalWrite(IN2,HIGH);
LOW LOW Stop (coast freely) digitalWrite(IN1,LOW); digitalWrite(IN2,LOW);
HIGH HIGH Brake (locked stop) digitalWrite(IN1,HIGH); digitalWrite(IN2,HIGH);
Speed pin: analogWrite(ENA, 0–255)  |  0 = stopped   128 = half speed (50%)   255 = full blast (100%)
💻 Core Code — motorForward / motorBackward / motorStop
int ENA = 9; // Speed (PWM) int IN1 = 2; // Direction int IN2 = 3; // Direction void motorForward(int speed) { void motorBackward(int speed) { digitalWrite(IN1, HIGH); digitalWrite(IN1, LOW); digitalWrite(IN2, LOW); digitalWrite(IN2, HIGH); analogWrite(ENA, speed); // 0-255 analogWrite(ENA, speed); } } void motorStop() { digitalWrite(IN1, LOW); digitalWrite(IN2, LOW); analogWrite(ENA, 0); } // In loop(): motorForward(128); delay(3000); // 50% speed for 3 sec motorStop(); delay(1000); motorBackward(255); delay(3000); // 100% speed backward motorStop(); delay(1000);

💡 My Activity Today — Wire it Up and Spin!
Wiring Diagram: 9V Battery (+) ---------> L298N +12V terminal 9V Battery (-) ---------> L298N GND terminal Arduino GND ---------> L298N GND terminal (SHARED GROUND!) L298N ENA ---------> Arduino Pin 9 (PWM speed) L298N IN1 ---------> Arduino Pin 2 (direction) L298N IN2 ---------> Arduino Pin 3 (direction) DC Motor wire A ---------> L298N OUT1 DC Motor wire B ---------> L298N OUT2 * Remove the ENA jumper cap on the L298N board so PWM works!
The Arduino GND and the battery GND MUST connect together. This is called a "shared ground" — without it, Arduino and the L298N cannot understand each other!
  1. Connect 9V battery (+) to L298N +12V terminal and battery (-) to L298N GND.
  2. Bridge Arduino GND to L298N GND (shared ground — this is critical!).
  3. Connect L298N ENA to Arduino Pin 9, IN1 to Pin 2, IN2 to Pin 3.
  4. Attach DC motor wires to L298N OUT1 and OUT2.
  5. Remove the ENA jumper cap from the L298N module.
  6. Upload the code — watch the motor go forward, stop, then backward!
  7. Wand Check: measure voltage on OUT1/OUT2 at PWM 0, 64, 128, 192, 255.

🔬 4 Experiments You Did Today!
A
Forward → Stop → Backward
Upload the basic motor code and watch the sequence.
Changing IN1/IN2 makes the motor flip direction — the H-Bridge reverses the current!
B
Potentiometer Speed Knob
Wire a pot to A0 and use map() to control speed.
Turning the knob changes the PWM duty cycle — like a gas pedal for your motor!
C
Wand Check — Measure Speed Voltage
Set Wand to DC 20V; probe OUT1 + OUT2 at PWM 0, 64, 128, 192, 255.
Voltage rises with PWM value — at 50% you see about 4–5 V; at 100% almost 9 V!
D
Missing Shared Ground
Disconnect Arduino GND from L298N and try to run the motor.
The motor won't respond — without a shared ground, the two circuits can't communicate!
