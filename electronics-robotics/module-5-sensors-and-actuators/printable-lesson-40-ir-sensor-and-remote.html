📚 Key Words — Learn These!
📷 IR
Infrared Light
Light below red on the spectrum — invisible to human eyes but cameras (and sensors) can see it! Remotes blast it 38,000 times per second.
📡 RX
IR Receiver
A tiny dome sensor (VS1838B) that catches infrared pulses and converts them into an electrical signal Arduino can read.
🔐 0x
Button Code (HEX)
Each remote button sends a unique pattern decoded into a hexadecimal number (e.g. 0xE916FF00). Like a fingerprint for every button!
🔄 ↻
Toggle
Switching something on then off then on with the same button press. Each press flips the state: on = !on.
💡 How IR Remote Control Works — Invisible Morse Code!
IR Receiver Pinout (VS1838B)
IR Receiver (front view) +-----+ / dome \ <-- aim at remote +---------+ | | | S G V i N C g D C Signal --> Arduino pin 11 GND --> Arduino GND VCC --> Arduino 5V
The 5-Step Signal Journey
  1. You press a button on the remote
  2. Remote flashes IR light pulses in a unique pattern
  3. IR receiver dome detects the pulses
  4. Converts pulses into an electrical signal
  5. Arduino decodes it into a button code number
Remote: [BTN 1] --> IR flash: ON-OFF-ON-ON...
Sensor: detects pulses --> signal pin LOW
Arduino: decodes --> 0xE916FF00 (Button 1!)
Every remote button sends a different blink pattern — like Morse code with invisible light at 38,000 blinks per second!
🔬 Step 4: Decode Your Remote — The Code-Breaker Sketch
🔐 Decoder Sketch
#include <IRremote.hpp> int irPin = 11; void setup() { Serial.begin(9600); IrReceiver.begin(irPin); } void loop() { if (IrReceiver.decode()) { Serial.print("Code: 0x"); Serial.println( IrReceiver.decodedIRData .decodedRawData, HEX); IrReceiver.resume(); } }

💡 My Activity Today — Remote-Controlled LED System!
Arduino Wiring: IR Receiver Signal --> Pin 11 (yellow wire) IR Receiver GND --> GND (black wire) IR Receiver VCC --> 5V (red wire) Red LED + 330ohm --> Pin 5 (red wire) Green LED + 330ohm --> Pin 6 (green wire) Blue LED + 330ohm --> Pin 7 (blue wire) All LED cathodes --> GND Key code in sketch (replace with YOUR codes!): #define BUTTON_1 0xE916FF00 // toggles Red LED #define BUTTON_2 0xE619FF00 // toggles Green LED #define BUTTON_3 0xF20DFF00 // toggles Blue LED #define BUTTON_PWR 0xBA45FF00 // all LEDs OFF
Replace the 0x... codes above with the ones YOU decoded from YOUR remote in Step 4!
Steps to Build:
  1. 1Install the IRremote library: Sketch > Include Library > Manage Libraries, search "IRremote" by shirriff/ArminJo.
  2. 2Wire the IR receiver: Signal → pin 11, GND → GND, VCC → 5V.
  3. 3Upload the Decoder Sketch (Side A). Open Serial Monitor. Press each button and write down the codes.
  4. 4Wire three LEDs (red/green/blue) each with a 330Ω resistor to pins 5, 6, and 7.
  5. 5Upload the LED Control Sketch with YOUR button codes. Press 1 = red, 2 = green, 3 = blue, Power = all off!
  6. 6Wand Check: multimeter set to DC Volts, black probe to GND, red probe to pin 11. Idle = ~5V. Hold button = ~3–4V average.
