📚 Key Words — Master These!
🌶 S
Sensor
A component that reads the world — temperature, light, motion, distance. It converts a physical measurement into an electrical signal your Arduino can understand.
⚙ A
Actuator
A component that does something in the physical world — a servo moves, an LED lights up, a buzzer beeps. Actuators turn electrical signals into action!
🔄 I2C
I2C Protocol
A communication method that uses only 2 wires (SDA + SCL) to let the Arduino talk to the LCD display. Many devices can share the same two wires!
📈 T
Threshold
A trigger value you set in code. If temperature goes above 28C, do something. If distance drops below 15cm, react! Thresholds make systems smart.
