📚 Key Words — Learn These!
🪆 Ω
Resistor
A tiny striped cylinder that limits how much current flows. It is the LED's bodyguard — stops it from getting too much electricity!
🔌 Ω
Ohm (Ω)
The unit we use to measure resistance. Named after Georg Simon Ohm, a German physicist who cracked the code in 1827!
🌈 #
Color Code
The secret spy code painted on every resistor as colored bands. Read the bands to find the resistance value — no wand needed!
📏 %
Tolerance
How close to perfect a resistor is allowed to be. Gold band = ±5% error is fine. Real resistors are never exactly right — and that's OK!
🏢 The Hallway Analogy — Resistance Slows the Flow!

🔬 4 Things You Tried Today!
A
Brightness Experiment
Swap resistors from 100Ω to 10kΩ in your LED circuit.
More resistance = less current = dimmer LED. The resistor controls brightness!
B
Decoder Card
Write out the color-to-number table on paper.
You decoded 3 real resistors: 1kΩ, 47kΩ, and 220Ω — like a spy!
C
Wand Check
Measure all 5 resistors with the multimeter (set to Ω symbol).
The Wand shows the REAL value. Color code gives the approximate value. Both are right!
D
Pencil Trick Bonus!
Draw a thick and a thin pencil line. Measure each with the Wand.
Graphite conducts! Thicker line = less resistance. Thin line = more resistance!
🏠 Try at Home: Pencil Resistor!
Draw a long, thick line with a soft pencil on plain paper. Put your multimeter probes on each end.

Now draw a thin, short line and measure again. Which has more resistance? 🔬
