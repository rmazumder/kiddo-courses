📚 All 8 Components You Mastered This Module!
⚡
Resistor
Slows current down. Color bands = value in Ohms (Ω).
💡
LED
Lights up when current flows. Long leg = positive (+).
🔋
Capacitor
Stores electrical charge. Like a tiny rechargeable tank.
⏬
Diode
One-way door for electricity. Stripe = negative end.
🔮
Transistor
Tiny switch + amplifier. Base, Collector, Emitter.
🔄
Switch & Potentiometer
Control the flow. Pot = adjustable resistance knob.
🔔
Buzzer
Turns electricity into sound. Active buzzer needs DC.
🔧
Multimeter (Wand)
Measures Voltage, Resistance, Continuity. Your best tool!

🛠 Build Steps Summary
1
Plan the Quiz (10 min)
Write 5 questions on the left side of the cardboard, scramble the answers on the right side. Use the age-appropriate question ideas from your lesson sheet.
+30 XP
2
Make Connection Points (10 min)
Push a brass paper fastener through each question dot and answer dot. Bend legs flat on the back. These are the contact points players touch.
+20 XP
3
Wire the Correct Pairs (10 min)
On the back, connect each question fastener to its correct answer fastener with wire or foil. Use tape to keep wires flat and separate.
+30 XP
4
Wand Check — Test Every Connection! (10 min)
Set Wand to continuity mode. Touch correct pairs — BEEP means the wire works. Touch wrong pairs — silence means no connection. Test all 5!
+50 XP
5
Build the Indicator Circuit (15 min)
9V(+) → PROBE 1 → Q-fastener → hidden wire → A-fastener → PROBE 2 → 330Ω → GREEN LED → 9V(−). Correct match = LED lights!
+40 XP
6
Wand-Assisted Debugging (5 min)
LED not lighting? Check battery voltage (DC Volts). Test probe wires (continuity). Check LED polarity — long leg is +. Debugging is solving, not failing!
+30 XP
7
Decorate! (5 min)
Draw a title — "ELECTRONICS QUIZ MASTER!" Add funny component drawings, color-code question and answer areas, and add a scoreboard. Make it yours!
+20 XP
✓
Final Assessment (bonus!)
Name all components. Answer the 6 concept check questions. Use the Wand to measure a battery, a resistor, and test a wire with continuity mode.
+50 XP
