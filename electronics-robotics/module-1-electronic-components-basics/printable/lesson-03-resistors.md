## What is a Resistor?

![Hallway Traffic Controller](../images/lesson-03-hallway-traffic-controller.png)

**A resistor is a speed bump for electrons:**

- **Low resistance** = Wide hallway = Fast flow
- **High resistance** = Narrow hallway = Slow flow
- **No resistance** = Short circuit = BAD! ⚠️

---

## Reading Resistor Color Codes

![Color Band Chart](../images/lesson-03-color-band-chart.png)

**A 4-band resistor tells you the resistance value:**

Band 1 + Band 2 = First two digits
Band 3 = Multiplier (×10, ×100, ×1000, etc.)
Band 4 = Tolerance (how accurate)

**Example:** Brown-Black-Red-Gold = 10 × 100 = 1000Ω ±5%

| Color | Digit | Multiplier |
|-------|-------|-----------|
| Black | 0 | ×1 |
| Brown | 1 | ×10 |
| Red | 2 | ×100 |
| Orange | 3 | ×1000 |
| Yellow | 4 | ×10K |
| Green | 5 | ×100K |
| Blue | 6 | ×1M |
| Violet | 7 | |
| Gray | 8 | |
| White | 9 | |
| Gold | — | ÷10 |

## Resistor Bodyguard Experiment

![LED Brightness](../images/lesson-03-led-bodyguard-brightness.png)

**Different resistors = Different LED brightness:**

| Resistor | Current | Brightness |
|----------|---------|-----------|
| 220Ω | More | Very bright |
| 470Ω | Medium | Medium |
| 1KΩ | Less | Dim |
| 10KΩ | Much less | Almost off |

**Rule:** Bigger resistor → Less current → Dimmer LED

| Value | Color Code | Use |

| 220Ω | Red-Red-Brown | LED protection (bright) |
| 330Ω | Orange-Orange-Brown | LED protection (medium) |
| 470Ω | Yellow-Violet-Brown | General purpose |
| 1KΩ | Brown-Black-Red | General purpose |
| 10KΩ | Brown-Black-Orange | Pull-ups, sensors |

## Real-World Uses

- 💡 **LED current limiting** — Resistor acts as bodyguard
- 📊 **Sensor circuits** — Voltage dividers
- 🎚️ **Volume control** — Variable resistor (potentiometer)
- 🌡️ **Temperature sensors** — Thermistors change with heat

---

## Quick Quiz

**Q1:** What does a resistor do?
**A:** It slows down current (limits the flow of electrons).

**Q2:** How do you read a 4-band resistor?
**A:** Bands 1-2 = digits, Band 3 = multiplier, Band 4 = tolerance.

**Q3:** Why is a resistor called an LED "bodyguard"?
**A:** Because it protects the LED from too much current, keeping it safe.
