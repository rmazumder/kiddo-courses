## 🐍 Python Cheat Sheet for Raspberry Pi GPIO

### Import & Setup:

```python
import RPi.GPIO as GPIO       # Import GPIO library
import time                   # Import time for delays

GPIO.setmode(GPIO.BCM)        # Use BCM pin numbering
GPIO.setwarnings(False)       # Suppress warnings (optional)
```

### Setting Up Pins:

```python
# OUTPUT pins (LEDs, buzzers, motors)
GPIO.setup(17, GPIO.OUT)

# INPUT pins (buttons, sensors)
GPIO.setup(18, GPIO.IN)

# INPUT with pull-down resistor (button default = LOW)
GPIO.setup(18, GPIO.IN, pull_up_down=GPIO.PUD_DOWN)

# INPUT with pull-up resistor (button default = HIGH)
GPIO.setup(18, GPIO.IN, pull_up_down=GPIO.PUD_UP)
```

### Controlling Outputs:

```python
GPIO.output(17, GPIO.HIGH)    # Turn ON (3.3V)
GPIO.output(17, GPIO.LOW)     # Turn OFF (0V)
GPIO.output(17, True)         # Same as HIGH
GPIO.output(17, False)        # Same as LOW
GPIO.output(17, 1)            # Same as HIGH
GPIO.output(17, 0)            # Same as LOW
```

### Reading Inputs:

```python
state = GPIO.input(18)        # Read pin (returns HIGH or LOW)

if GPIO.input(18) == GPIO.HIGH:
    print("Button pressed!")
```

### PWM (Pulse Width Modulation):

```python
# Create PWM on pin 18, frequency 1000 Hz
pwm = GPIO.PWM(18, 1000)

pwm.start(50)                # Start with 50% duty cycle
pwm.ChangeDutyCycle(75)      # Change duty cycle (0-100)
pwm.ChangeFrequency(2000)   # Change frequency
pwm.stop()                   # Stop PWM
```

### Timing:

```python
time.sleep(1)                 # Wait 1 second
time.sleep(0.5)               # Wait half a second
time.sleep(0.001)             # Wait 1 millisecond

start = time.time()           # Record start time
# ... do something ...
elapsed = time.time() - start # Calculate elapsed time
```

### Cleanup:

```python
GPIO.cleanup()                # Reset ALL pins
GPIO.cleanup(17)              # Reset only pin 17
GPIO.cleanup([17, 18, 27])   # Reset specific pins
```

### Complete Template:

```python
#!/usr/bin/env python3
import RPi.GPIO as GPIO
import time

# Pin definitions
LED = 17
BUTTON = 18

# Setup
GPIO.setmode(GPIO.BCM)
GPIO.setup(LED, GPIO.OUT)
GPIO.setup(BUTTON, GPIO.IN, pull_up_down=GPIO.PUD_DOWN)

try:
    while True:
        if GPIO.input(BUTTON) == GPIO.HIGH:
            GPIO.output(LED, GPIO.HIGH)
        else:
            GPIO.output(LED, GPIO.LOW)
        time.sleep(0.1)

except KeyboardInterrupt:
    print("Stopped!")

finally:
    GPIO.cleanup()
```

## ⚡ Component Reference Card

### LED (Light Emitting Diode):
```
Long leg (+) = Anode (positive) → connects to GPIO through resistor
Short leg (-) = Cathode (negative) → connects to GND
ALWAYS use a 220Ω resistor!
```

### Resistor Values Used:
```
220Ω  → For LEDs (current limiting)
1KΩ   → Voltage divider (HC-SR04)
2KΩ   → Voltage divider (HC-SR04)
10KΩ  → Pull-down for buttons (optional with PUD_DOWN)
```

### Breadboard:
```
Power rails (+ and -) = Connected horizontally
Main area = Connected vertically (5 holes per group)
Center gap = Separates left (a-e) from right (f-j)
```

### DHT11 Temperature/Humidity Sensor:
```
Pin 1: VCC (3.3V or 5V)
Pin 2: DATA (to GPIO, e.g., GPIO 4)
Pin 3: Not used
Pin 4: GND
Library: adafruit-circuitpython-dht
```

### HC-SR04 Ultrasonic Distance Sensor:
```
VCC:  5V power
TRIG: Output from Pi (triggers the ping)
ECHO: Input to Pi (receives the echo) ⚠️ NEEDS VOLTAGE DIVIDER!
GND:  Ground

Voltage Divider: ECHO → [1KΩ] → GPIO → [2KΩ] → GND
Distance formula: (pulse_duration × 34300) / 2
```

### Buzzer:
```
Active Buzzer: Just power it = BEEP (one tone)
Passive Buzzer: Use PWM to control frequency (play music!)
(+) leg → GPIO pin
(-) leg → GND
```
