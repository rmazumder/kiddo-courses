### Multiple Choice Answers:

| Question | Answer | Explanation |
|----------|--------|-------------|
| 1 | **B)** Command Line Interface | The text-based way to interact with your computer |
| 2 | **C)** Prints text to the screen | `echo "Hello"` displays "Hello" |
| 3 | **C)** Your logged-in username | It tells you who you are! |
| 4 | **B)** Ctrl + C | The emergency stop button! |
| 5 | **C)** Auto-completes file and folder names | Your best friend for avoiding typos! |
| 6 | **B)** Print Working Directory | Shows your current location |
| 7 | **C)** Hidden files (starting with a dot) | Files like `.bashrc` are hidden by default |
| 8 | **C)** Goes up one level to the parent directory | `..` always means "parent" |
| 9 | **C)** Goes to your home directory | `~` is the shortcut for home |
| 10 | **B)** `/home/alex/Pictures` | `..` goes up to `/home/alex`, then into `Pictures` |
| 11 | **B)** `mkdir` | Make Directory = make a folder! |
| 12 | **C)** Creates a new empty file | `touch` creates files (or updates timestamps) |
| 13 | **B)** cp copies, mv moves | Copy = duplicate, Move = relocate |
| 14 | **B)** `cp -r folder backup` | `-r` (recursive) is required for folders! |
| 15 | **C)** Permanently deletes with no undo | Always double-check before using `rm`! |
