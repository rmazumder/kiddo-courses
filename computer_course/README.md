# 🖥️ Command Quest: The Computer Explorer's Adventure Course

## 🌟 Welcome, Future Computer Wizard!

Have you ever wondered how computers **really** work? Or wished you could control a computer using secret text commands — like a hacker in a movie? Well, you're about to learn exactly that!

**Command Quest** is your adventure through the amazing world of computers. You'll start by understanding what makes a computer tick, then learn to use the **Terminal** — a powerful tool that lets you talk directly to your computer using typed commands instead of clicking buttons.

By the end of this course, you won't just *use* a computer — you'll **command** it. 🧙‍♂️

---

## 🎯 Who Is This Course For?

- **Ages:** 8–14 years old
- **Experience needed:** None! Just curiosity and a computer to practice on
- **Works on:** Mac, Windows (with WSL or Git Bash), or Linux

## 📖 How to Use This Course

1. **Go in order** — Each module builds on the previous one
2. **Practice everything** — Type the commands yourself! Reading isn't enough
3. **Don't fear mistakes** — The terminal won't break your computer (we'll show you safe commands)
4. **Take the quizzes** — They help you remember what you learned
5. **Have fun!** — This is an adventure, not homework

---

## 🛠️ What You'll Need

- A computer (Mac, Windows, or Linux)
- **Mac users:** Open the "Terminal" app (search for it in Spotlight)
- **Windows users:** Install [Git Bash](https://git-scm.com/downloads) or use WSL
- **Linux users:** Open your terminal emulator
- A sense of adventure! 🚀

---

## 👨‍👩‍👧‍👦 Note for Parents & Teachers

This course teaches foundational computer literacy and command-line skills through gamified, age-appropriate lessons. All commands taught are safe and non-destructive. Children are encouraged to practice in a dedicated folder to avoid accidentally modifying important files.

**Estimated completion time:** 6–10 hours (self-paced)
