## 🎮 Command Cheat Sheet — Quick Reference

Here's every command you learned, all in one place!

### Navigation
| Command | What It Does |
|---------|-------------|
| `pwd` | Print current directory (Where am I?) |
| `ls` | List files and folders |
| `ls -la` | List everything with details |
| `cd folder` | Change into a folder |
| `cd ..` | Go up one level |
| `cd ~` | Go to home directory |

### File Management
| Command | What It Does |
|---------|-------------|
| `mkdir folder` | Create a folder |
| `mkdir -p a/b/c` | Create nested folders |
| `touch file.txt` | Create an empty file |
| `cp src dst` | Copy a file |
| `cp -r src dst` | Copy a folder |
| `mv src dst` | Move or rename |
| `rm file` | Delete a file (permanent!) |
| `rm -r folder` | Delete a folder (permanent!) |
| `rm -i file` | Delete with safety prompt |

### Viewing Files
| Command | What It Does |
|---------|-------------|
| `cat file` | Show entire file |
| `cat -n file` | Show file with line numbers |
| `less file` | Browse file (q to quit) |
| `head file` | Show first 10 lines |
| `head -n 5 file` | Show first 5 lines |
| `tail file` | Show last 10 lines |
| `tail -n 5 file` | Show last 5 lines |

### Searching
| Command | What It Does |
|---------|-------------|
| `find . -name "*.txt"` | Find files by name |
| `find . -type d` | Find only directories |
| `grep "word" file` | Search for text in a file |
| `grep -r "word" folder` | Search recursively |
| `grep -i "word" file` | Case-insensitive search |
| `grep -n "word" file` | Show line numbers |
| `grep -c "word" file` | Count matches |

### Pipes & Redirects
| Command | What It Does |
|---------|-------------|
| `cmd > file` | Redirect output to file (overwrite) |
| `cmd >> file` | Append output to file |
| `cmd1 \| cmd2` | Pipe output to next command |
| `sort` | Sort lines alphabetically |
| `uniq` | Remove duplicate lines |
| `wc -l` | Count lines |
| `wc -w` | Count words |

### Text Editing
| Command | What It Does |
|---------|-------------|
| `nano file` | Open file in nano editor |
| `Ctrl+O` | Save in nano |
| `Ctrl+X` | Exit nano |
| `Ctrl+W` | Search in nano |
| `vim file` | Open file in vim |
| `Esc` then `:wq` | Save and quit vim |
| `Esc` then `:q!` | Quit vim without saving |

### Utilities
| Command | What It Does |
|---------|-------------|
| `echo "text"` | Print text to screen |
| `date` | Show current date/time |
| `whoami` | Show your username |
| `clear` | Clear the screen |
