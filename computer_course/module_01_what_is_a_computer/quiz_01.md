## ❓ Question 10: Tree Reader

Look at this file tree:

```
📁 home
└── 📁 jordan
    ├── 📁 Music
    │   └── 🎵 song.mp3
    └── 📁 Photos
        └── 🖼️ beach.jpg
```

What is the **absolute path** to `beach.jpg`?

- A) `Photos/beach.jpg`
- B) `/jordan/Photos/beach.jpg`
- C) `/home/jordan/Photos/beach.jpg`
- D) `/home/Photos/beach.jpg`

## 🔑 Answer Key

<details>
<summary>🔐 Click to reveal all answers! (Try to answer everything first!)</summary>

| Question | Correct Answer | Explanation |
|----------|---------------|-------------|
| 1 | **B)** Input → Process → Output | Every computer follows this pattern! |
| 2 | **C)** Google Chrome | Chrome is a program (software). The others are physical things (hardware). |
| 3 | **C)** Processes information — it's the brain | CPU = Central Processing Unit = the computer's brain! |
| 4 | **A)** RAM is faster, but hard drive keeps files when off | RAM is like a desk (temporary workspace), hard drive is like a filing cabinet (permanent storage). |
| 5 | **D)** Speakers | Speakers send information OUT to you (sound). The others send information IN. |
| 6 | **C)** A video file | The `.mp4` extension means it's a video file. |
| 7 | **C)** `/home/sam/Documents` | The parent directory is the folder that directly contains the file. |
| 8 | **A)** True | Folders inside folders is how the file system tree works! |
| 9 | **C)** The parent directory (one level up) | `..` always means "go up one level." |
| 10 | **C)** `/home/jordan/Photos/beach.jpg` | Start at root, trace through each folder to the file. |
