## What This Module Is About

Now that you know what Bitcoin is and how trading works, it is time to learn the MOST important skill any investor can have: **reading charts**. Charts are like maps for investors -- they show you where the price has been and help you think about where it might go.

In this module, you will learn to read line charts, discover the magic of candlestick charts (they look like candles on a birthday cake!), and find invisible walls called support and resistance that prices bounce off of.

## Module Goals

By the end of this module, you will:

- Read and draw line charts showing price over time
- Understand candlestick charts and what each part means

- Spot basic patterns that experienced traders look for

## Parent/Instructor Notes

- This module is more visual than Module 1 -- lots of drawing and chart reading
- **Graph paper** makes the activities much easier but is not required
- Ages 10--11: Focus on the visual patterns and skip the math. Drawing charts by hand is the best learning tool
- Ages 12--14: Encourage them to look at real Bitcoin charts online (TradingView offers free charts) and compare to what they draw
- Some kids may get frustrated with chart drawing at first -- remind them that even professional traders started this way
- Track XP visually -- consider adding to the same poster or notebook from Module 1
